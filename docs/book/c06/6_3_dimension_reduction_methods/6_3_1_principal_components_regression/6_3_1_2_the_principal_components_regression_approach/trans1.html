6.3.1.2 주성분 회귀 접근법 (The Principal Components Regression Approach)

The principal components regression (PCR) approach involves constructing the first $M$ principal components, $Z_1, \ldots, Z_M$, and then using these components as the predictors in a linear regression model that is fit using least squares. The key idea is that often a small number of principal components suffice to explain most of the variability in the data, as well as the relationship with the response. In other words, we assume that the directions in which $X_1, \ldots, X_p$ show the most variation are the directions that are associated with $Y$. While this assumption is not guaranteed to be true, it often turns out to be a reasonable enough approximation to give good results. 주성분 회귀(principal components regression, PCR) 접근법은 기저 데이터의 처음 $M$개 주성분 $Z_1, \ldots, Z_M$을 수립 구축하는 과정을 거친 뒤, 도출된 이 성분들을 최소 제곱법(least squares)으로 적합된 선형 회귀 모델의 예측 변수로 활용하는 체계를 포함합니다. 주요 핵심 아이디어는 소수의 주성분만으로도 데이터 내의 분산 변동성 대부분은 물론, 반응 변수와의 관계성까지 충분히 다 설명해 낼 수 있다는 점입니다. 다시 말해서, 우리는 $X_1, \ldots, X_p$ 인자들이 가장 막대한 변동성을 보여주는 벡터 방향이 곧 반응 변수 $Y$와 연관된 방향일 것이다 라고 가정합니다. 비록 이 가정이 항상 절대적 참이 아닐 수도 있으나, 훌륭한 산출 결과를 타전하기에 꽤나 합리적이고 타당한 근사치임이 종종 입증됩니다.

If the assumption underlying PCR holds, then fitting a least squares model to $Z_1, \ldots, Z_M$ will lead to better results than fitting a least squares model to $X_1, \ldots, X_p$, since most or all of the information in the data that relates to the response is contained in $Z_1, \ldots, Z_M$, and by estimating only $M \ll p$ coefficients we can mitigate overfitting. In the advertising data, the first principal component explains most of the variance in both pop and ad, so a principal component regression that uses this single variable to predict some response of interest, such as sales, will likely perform quite well. 만약 앞서 언급한 PCR 밑바탕의 내재 가정이 참으로 성립 유지된다면, $Z_1, \ldots, Z_M$ 단위들에 최소 제곱법 기반 모델을 적합시키는 것은 원래의 $X_1, \ldots, X_p$ 수량에 모델을 적합시키는 것보다 더 명백히 나은 결과를 도출할 것입니다. 왜냐하면 반응 변수와 연계된 데이터의 파장 정보 대부분 혹은 그 전체 일체가 이 $Z_1, \ldots, Z_M$ 궤도 내에 농축되어 포섭되어 있고, 더불어 원시 변수보다 훨씬 적은 수치인 오직 $M \ll p$ 개의 계수만을 추정함으로써 모델의 잠재적 과적합(overfitting) 부작용을 유의미하게 경감시킬 수 있기 때문입니다. 앞선 광고 데이터 분석 예시에서, 제 1 주성분 하나가 투입 지표 pop 및 소진 재원 ad 요인 양 단면의 압도적 분산 대다수를 대변 설명하므로, 이 단일 축 변수 하나만을 덩그러니 사용해 타겟 sales 와 같은 임의 관측 관심사의 반응값을 도출 예측하는 단일 주성분 회귀 모델은 아주 탁월하게 구동 성능을 뽐낼 가능성이 매우 높습니다.

In contrast, PCR will tend to do well in cases when the first few principal components are sufficient to capture most of the variation in the predictors as well as the relationship with the response. We note that even though PCR provides a simple way to perform regression using $M < p$ predictors, it is not a feature selection method. This is because each of the $M$ principal components used in the regression is a linear combination of all $p$ of the original features. For instance, in (6.19), $Z_1$ was a linear combination of both pop and ad. One can even think of ridge regression as a continuous version of PCR![8] 원본 핵심 특징 피처 초기 자원들의 적은 소규모 선별 세트 부분 집합에만 전면 의존 구동하는 편향된 압축 제한 모형 단일 시스템 체계의 개발이나 진척 단면 수립 도래 단계 결과로는 절대 연결 산출 및 전개 귀결되지 않습니다. 오로지 바로 이러기 같은 고정 의미 관점 진영 체계 하에서 통계학상, PCR 구조 전개 태세 기법 구조는 상대방 위상인 라쏘(lasso) 추정 구조법보다는 역으로 되려 압박 능선 회귀 단면 체제 양상과 훨씬 더 밀접한 구조적 친밀 다변 인척 지표 연관성 상관 동위 결속 관계를 맺고 지닙니다. 사실 파헤쳐 들여 볼수록 단연코 증명 가능하듯, PCR 방식론 기저와 능선 회귀 기저는 수리 모델상 통계적으로 가히 매우 이질 치밀 동질적으로 가깝게도 밀접 친밀 긴밀하게 연결 연관 수립된 기저 친족적 밀월 지배 체계입니다. 우리는 심지어 극단적으로 지표 상 산출된 정규화 위상 양태 거시적 능선 회귀 수리 기법 구조 자체를, 단절 한계 이면 없이 다원 전개되는 PCR 연쇄 체제 기법 시스템의 하나의 “연속적 연속 동조화 전개형 파생 교차 버전(continuous version)”으로 도출 간주하고 단면 생각 체계를 확장해 동일시해 판단할 수도 있습니다![8]

In PCR, the number of principal components, $M$, is typically chosen by cross-validation. The results of applying PCR to the Credit data set are shown in Figure 6.20; the right-hand panel displays the cross-validation errors obtained, as a function of $M$. On these data, the lowest cross-validation error occurs when there are $M = 10$ components; this corresponds to almost no dimension reduction at all, since PCR with $M = 11$ is equivalent to simply performing least squares. 구동되는 PCR 체계 전제 하에서, 취합 점유할 누적 편입 대상 주성분 요소 할당 개수 척도 수량인 차원 스케일 $M$ 은 전반적 단면상 으레 흔히 보편 전형적으로 널리 쓰이는 표본 교차 검증(cross-validation) 평가 수칙에 의거해 최종 선별 지목 및 적합 선택 결정 조율됩니다. 표본 단면 데이터 축 Credit 데이터 기점 세트에 이 단일 PCR 방법 기법 척도를 직접 응용 부착 거점 적용한 결과 산출 전개 양태 양상은 상기 단락 부근의 그림 표면 6.20 위 평면 선상에 공표 표기 표시되어 있습니다; 해당 패널 사진 화면 중 시면을 기점으로 한 가로 우측 오른쪽 영역 동면 패널 화면 단면이, 지정 할당 척도 차원값 $M$ 증폭 수위 인자 변화 대응 함수 전개 곡선으로서 전면 측정 도달로 획득된 잉여 교차 검정 체제 편차 무리 오차 에러 곡선율(cross-validation errors obtained) 분산을 조목조목 전시 전개 도해해 대변 보여줍니다. 위로 관측 분석 대상 적시 도마에 오른 통계 지표 단면 데이터 집단 시스템상에서, 가장 최하 바닥을 친 절저 최저 극저점(lowest) 산포 교차 오차 검증 범위 구간 낙하 에러 발생 현상은 도합 주성분 요소 분량이 $M = 10$ 개 단위(components) 뭉치로 꽉꽉 톱니바퀴 조립 투입 축적 구성 구비 존재할 때야 비로소 실재 포착 단면 도출 귀결 발생합니다; 하지만 이는 사실 기저적으로 치명적인 역설 단서로 그 어느 수준 차원 절묘한 축소 과정(dimension reduction mechanism) 체계 요소도 절대 거의 거시 단면 전혀 털끝조차 사실상 아예 하나도 전무 이뤄 내지 연출해 도출 달성하지 못한 본질 전 상태 원위치 사안 현실에 직접 고스란히 완전 대응 일치 매칭 상응(corresponds) 대변합니다, 왜냐하면 투입 $M = 11$ 척도 범위를 보유 구비한 막장 PCR 구동 방식은 애초 시발점의 무계획 그저 가장 기본 단순하고 평범 순진무구 오리지날 변수 최소 제곱법 단일 기법을 백지상태로 단순 전면 순진하게 수행 파생(simply performing least squares) 치르는 도출 행위 일체 과정과 수학 물리 현상 기하학적으로 일치 매칭 완벽 대변 등치 상응하여 절대(equivalent to) 똑같기 때문입니다.

When performing PCR, we generally recommend standardizing each predictor, using (6.6), prior to generating the principal components. This standardization ensures that all variables are on the same scale. In the absence of standardization, the high-variance variables will tend to play a larger role in the principal components obtained, and the scale on which the variables are measured will ultimately have an effect on the final PCR model. However, if the variables are all measured in the same units (say, kilograms, or inches), then one might choose not to standardize them. PCR 궤도 체계를 가동 구동 수리 실행 적합 전개할 때, 우리는 대개 언제나 일선 본체 주성분 무리 인자 요인망들을 초기 시도 단계부터 최초 형성 파생 축적 생성 도출 발탁 추출해 내기 이전에 앞서 미리 사전 수차 조치 선행 기저 기반 작업으로, 수칙 공식 도해 단락 (6.6) 규약을 부착 활용하여 각 투입 진영의 다원 예측 변수 요소 요인 피처 척도들을 일괄 모조리 표준화(standardization) 가공 공정 전처리 정화 조치 제어 취급 통제할 것을 가급적 도의적으로 전면 일반적으로 폭넓게 거시 권장 권고 시사 타전 추천 제안 가이드 척도로 부 여 하 고 제 시 주 목 합 니 다(recommend). 이 필수 거시 표준화 절차 기법은 모든 분별 다채로운 입력 변수 통계 구성 지표 요소 군단들이 결점 무탈히 서로 동일한 동등 격차 등률 측정 척도 스케일 크기 분량 등급(same scale) 상에 올바르게 균형 동일선 안착 위치 진입하도록 보장 확증 공시 확언합니다. 만약 이런 거시 전처리 사전 제어 무장된 표준화 확립 시스템이 전면 공백 부재(absence)하여 누락되는 환경 하 사단에서는, 고분산 즉 높은 오차 진폭의 크고 거대한 자체 엇갈림 오차 파동 분산(high-variance) 스케일을 소유 지니고 우쭐대는 유별 거대 수리 변수 녀석 군단 무리들이 막상 단면 수합 획득된 공고 파생 도출 수립 파생된 핵심 최상 차순위 요인 단일 주성분 지수 뼈대 객체들 안에서 편향 집중되어 월권 부당 상대적으로 극대화 비대 차대 엄청나게 거대한 지배 우위 독점 파장 장악 역할(larger role) 수행 비중 파워 단면 압도를 도래 행사하려는 경향과 편파적 조종 편향 집중 현상 유도를 편향 초래 발휘 꾀하게 될 것이며, 이로 인해 애초 오리지널 투입 통계 변수군들이 원천 자체적으로 측정 계산 단락 적발 파생 수립 체계 산출 관측되었던 그 초기 도량 단위 오만 자체의 스케일 수치 격차가 결국 종국에 가서는 산출 수렴된 최후 막단 종결 산출 마지막 무장 도출 지표 파생된 단층 파이널 분석 최후 PCR 기반 모델 단락의 완성품 품질 전체 전면 형상 도출 수지에까지 전방위 결함 훼손 중대 여파 효과 영향 간섭(an effect on…) 작용 파문을 근본적으로 지대 불가피하게 파생 유발 전면적으로 지독히 연출 행사 미치게 될 것입니다. 그러나 막상 오직 단 조건부 예외(However)적 가정 하나로, 만약 투입 참여 데이터들의 변수 요소 단위 척도 군단들이 몽땅 전부 다 모조리 (가령 다들 킬로그램 단일 중량이라든가 혹은 다 길이 인치 식의) 일체 동일 통일 동등한 단일 도량형 단위 체제 포맷 체계 격식 포맷 수치 단위(say, kilograms, or inches) 기저 단추망 내부 자체에서 일괄 측량 조사 판별 측정 수합 적출 확보 수집 형성 관측되어 마련된 상태 여건 체계라면, 굳이 분석 기법 사용 도출자 입장에서 그런 상태의 오리지날 데이터 무리들에는 모종의 번거로운 가공 규격 압박 무장 적합 환산 억제 차원 인위 지배 관섭 압력 제재 편입 인위 편제 기공 압축 통제 가동을 포기하여 그것들을 표준화 적제화 표준 동기 조치 처리 적정 무장 적용시키지 않기로(not to standardize them) 고의 방조 자유 여지 용인 우회 면책 채택 타결 선택 결단(choose) 결정 내리게 됨을 용인할 수도 다분히 있습니다.
