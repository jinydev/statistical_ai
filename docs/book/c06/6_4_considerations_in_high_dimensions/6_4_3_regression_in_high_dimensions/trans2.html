6.4.3 고차원에서의 회귀 (Regression in High Dimensions)

It turns out that many of the methods seen in this chapter for fitting less flexible least squares models, such as forward stepwise selection, ridge regression, the lasso, and principal components regression, are particularly useful for performing regression in the high-dimensional setting. Essentially, these approaches avoid overfitting by using a less flexible fitting approach than least squares. 우리는 앞선 챕터들을 돌며 표준 최소 제곱 모델 방식이 가진 과도한 유연성 태생적 한계를 통제하기 위해 의도적으로 모델을 덜 유연하게(less flexible) 묶어두고 적합시켜 성과를 얻는 수많은 제어 테크닉들을 구경했습니다. 이때 배웠던 전진 단계 선택법(forward stepwise selection), 능선 회귀(ridge regression), 라쏘(lasso) 추론, 그리고 주성분 회귀(principal components regression)와 같은 통제 기법들이 뜻밖에도 방금 우리가 맞닥뜨린 치명적인 고차원 세팅 모의 분석 환경 문제를 돌파하고 회귀를 올바르게 도출 수행해 내는데 특히나 구세주같이 유용하게 작용한다는 놀라운 사실이 증명 밝혀졌습니다. 이 치유 분석 접근법들의 성공 본질 핵심은 이렇습니다: 이들은 통제 장치를 거쳐 단순 무식한 표준 최소 제곱법보다는 훨씬 덜 유연하고 다소 뻣뻣한 모형 제한 적합 방식을 시스템에 강제 구사함으로써, 앞서 경험한 과적합 궤도라는 끔찍한 예측 함정을 손쉽게 회피하고 비껴갑니다.

Figure 6.24 illustrates the performance of the lasso in a simple simulated example. There are $p = 20, 50, \text{or } 2,000$ features, of which 20 are truly associated with the outcome. The lasso was performed on $n = 100$ training observations, and the mean squared error was evaluated on an independent test set. As the number of features increases, the test set error increases. When $p = 20$, the lowest validation set error was achieved when $\lambda$ in (6.7) was small; however, when $p$ was larger then the lowest validation set error was achieved using a larger value of $\lambda$. In each boxplot, rather than reporting the values of $\lambda$ used, the degrees of freedom of the resulting lasso solution is displayed; this is simply the number of non-zero coefficient estimates in the lasso solution, and is a measure of the flexibility of the lasso fit. Figure 6.24 highlights three important points: (1) regularization or shrinkage plays a key role in high-dimensional problems, (2) appropriate tuning parameter selection is crucial for good predictive performance, and (3) the test error tends to increase as the dimensionality of the problem (i.e. the number of features or predictors) increases, unless the additional features are truly associated with the response. 시뮬레이션된 모의 예제를 토대로 라쏘(lasso) 통제 기법이 얼마나 훌륭한지 그림 6.24의 그래픽을 통해 함께 증명 탐구해 보겠습니다. 데이터상 특성 변수는 각기 다르게 $p = 20, 50$, 또 극단적인 수치인 $2,000$개로 할당되어 존재합니다. (이 중 실제 타겟 산출 예측 종점과 연결되어 참된 연관 반응을 하는 진짜 연관 변수 단위는 오직 20개뿐입니다.) 훈련 세트상 관측치는 딱 $n = 100$개만 띄워놓고 라쏘 모델을 훈련 적합 구동시켰으며, 모델의 신뢰성 검증 단계를 위해 격리되어 있던 분할 독립 테스트 세트를 따로 세팅해 평균 제곱 오차 테스트 지수를 도출했습니다. 모델 체적 예측에 삽입 끼워 넣는 이 특성 변수 객체들의 양적 규모 수가 점점 뻥튀기될수록 기조 측정 테스트 세트 오답 한계 오차 역치 수위도 함께 파멸적으로 덩달아 상승 치솟습니다. 단면 환경 $p = 20$인 기초 실험 조별 진영에선 (6.7) 페널티 통제 지수인 $\lambda$ 치수를 가장 적게 살짝만 걸어 뒀을 때 역설적으로 테스트 측정 최저 검증 세트 오차가 최고 우수 점수로 파생 도출 수렴되는 성공을 달성했습니다. 반대로 $p$ 볼륨 덩치가 훨씬 거대해진 다른 비교 관측 진영 무대에선 통제 규제 치수 $\lambda$ 값을 한참이나 높게 강하게 세팅하여 조였을 때 비로소 간신히 모델의 성능 안전판 마지노선인 최저 검증 세트 오차를 허덕이며 얻어낼 수 있었습니다. 덧붙여 각각의 상자 그림(boxplot) 도표 상 출력 화면에선 적용 편차 세팅 척도인 $\lambda$ 값을 거추장스럽게 구구절절 일일이 띄우기보단, 결과 종점 계산 도출된 라쏘 해 결괏값의 자유도(degrees of freedom) 척도 단위를 깔끔한 계산 판별 단위로 삼아 표기 나열하였습니다. 이 수치는 단순 라쏘 계산 결괏값 산출 단에서 유효 판별 치인 0이 아닌 수치로 적합 할당된 계수 추정치의 순수 개수를 직관적으로 나타내며 동시에 라쏘 모델이 얼마나 융통성 있게 헐렁한지 그 유연성에 대한 정도 파악 척도로써 작용합니다. 명실상부한 통계 지침서 그림 6.24는 여기서 매우 가치 있는 세 가지 핵심 철칙 기조 사항을 아주 명백히 일깨워 강조합니다. (1) 지독한 변동성 통제가 필수인 고차원 문제 한계 상황에서는 규제(regularization) 즉 축소(shrinkage) 기법 수단이 모델을 관통하는 가장 핵심 코어 중추 역할을 독차지합니다. (2) 최고 도출 적합의 우수한 성과 예측 성적을 도모 획득하려면 무엇보다 가장 적합한 튜닝 파라미터 규제 세팅 계수 치수 선정 통찰 능력이 무척이나 결정적이고 필수적입니다. 그리고 마지막으로 (3) 무턱대고 새로 구비 추가하려 드는 확장 특성 변수 성질 요소 객체들이, 처음부터 이 목표 종점 반응 도출과 확고한 실질 깊은 연관 관련성을 보유 연계 유지한 게 결코 이라면, 문제 환경의 거대 차원(즉, 쏟아져 들어온 특성 혹은 예측 변수들의 방대한 총 개수 부피 규모)이 무대 위에 무참히 쏟아져 늘어날수록 당연히 테스트 검증 객관 평가 오류 편차 수치 측정 치수도 치명적 우주 폭발하듯 끔찍히 거대하게 솟아버리는 최악의 패널티 악경향을 띱니다.

The third point above is in fact a key principle in the analysis of high-dimensional data, which is known as the curse of dimensionality. One might think that as the number of features used to fit a model increases, the quality of the fitted model will increase as well. However, comparing the left-hand and right-hand panels in Figure 6.24, we see that this is not necessarily the case: in this example, the test set $\text{MSE}$ almost doubles as $p$ increases from 20 to 2,000. In general, adding additional signal features that are truly associated with the response will improve the fitted model, in the sense of leading to a reduction in test set error. However, adding noise features that are not truly associated with the response will lead to a deterioration in the fitted model, and consequently an increased test set error. This is because noise features increase the dimensionality of the problem, exacerbating the risk of overfitting (since noise features may be assigned nonzero coefficients due to chance associations with the response on the training set) without any potential upside in terms of improved test set error. Thus, we see that new technologies that allow for the collection of measurements for thousands or millions of features are a double-edged sword: they can lead to improved predictive models if these features are in fact relevant to the problem at hand, but will lead to worse results if the features are not relevant. Even if they are relevant, the variance incurred in fitting their coefficients may outweigh the reduction in bias that they bring. 방금 명시한 이 세 번째 경고 조항이 사실상 통계업계 전반에서 전설처럼 일컬어지는 유명한 개념인 차원의 저주(curse of dimensionality) 로 불리는, 복잡 고차원 데이터 평가 분석 무대의 가장 뼈대 깊은 핵심 경계 철칙의 근원입니다. 모의 훈련 단계를 관찰하는 미숙한 관찰자는 대개 기초 모델 훈련단에 들이붓고 적합에 써먹어 끼워 다 구겨 넣는 특성 변수의 수치 개수가 많아질수록 산출 적합 모델의 완성된 예측 성능 품질 성과 역시 똑같이 거저 거저먹는 양 당연히 상승할 것이란 아둔한 착각에 빠지게 됩니다. 허나 그림 6.24 무대에 차려진 좌우 실험 단상 패널 기조를 냉정히 교차 비교 관측해 보면 이 일방적 철칙 망상이 결코 정답이 아님이 명백히 깨어집니다. 단적인 증거 단면으로 당장 이 해당 시뮬레이션 지표 무대에서만 보더라도 변수 특성 차수 투입 $p$ 규격 규모가 원 기준 20개에서 허황된 2,000개 수위로 극한으로 뛰어 모의 확산 증가할 때, 척도 판단 평가 모델 $\text{MSE}$가 미친 듯 추락하며 오차 편차가 거의 배 수준 치수로 치명 뻥튀기돼 버림을 아주 적나라하게 관전 확인할 수 있습니다. 보편 타당히 합리 상식적 기본 관측 모델 분석에서, 테스트 세트 종점 평가 오류 수치 한계를 착실하게 깎아 줄여 혜택을 도모해 준단 참된 긍정 순기능 관전 의미로, “원하는 목표 종점 파트 결과 반응 반응과 진짜 깊은 연관 관련성을 보유 연계 유지한 실질 유효 핵심 시그널 객체 신호 특성 확장 요소들을 발굴 편입시켜 추가 투입하는 조치는, 당연히 적합 완성된 모델 성능 수준 품질을 빛나게 우수히 개선 향상 끌어올려 도모해 나갈 것입니다”. 허나 그와 정반대로, 겉보기 구색만 끼운 목표치 반응 산출과 한치도 상관없는, 껍데기뿐인 허위 가짜 쓰레기 불순 소음(noise) 특성 방관 지표 요소들을 함부로 통제 불가하게 무차별 확장 대량 배정 모델에 추가 투입해 집어넣는 태만 방기 행위는 무조건 극심하게 파탄 최악의 비극 결과 전복 재앙을 유발해 적합 통제 진영 모델 품질 평가를 무참하게 오염 파괴 병들게 하여 결과적으로 테스트 최종 검증 세팅 단 평가 오류 수치를 가차 없이 끔찍한 막대 통제 붕괴 상승 악영향 악화 상승시키는 재앙으로 떨어집니다. 왜냐하면 이런 쓰레기 모델 소음 지수 가짜 불순 특성 편입 조치들이 도달 문제 체제 해결 차원 덩치 단계를 끝없이 팽창시켜 부풀리는 행위 자체가, 단 0.1%의 테스트 세트 오차율 한계 개선 감소 가능 최적 이점 잠재 긍정 요인 가능성마저도 깡그리 증발 없애 버린 채 오직 무참한 파탄 과적합 병폐 붕괴 악화 진입 모형 예측 지수 참사 파탄만을 조장 자극 모의 (심지어 겉으로는 쓸모없어 쓰레기인 이 거짓 허위 소음 특성 지표들조차도 편향 훈련 세팅 데이터 조작 눈속임 구조 무대 위에서는 아주 공교롭게 기막힌 조작 우연의 일치 기적 확률로 마치 긍정 연관 진단 산출 결론이 통과 도달하는 상관관계 환상 작용 환각 계산 오류 현상 때문에 버젓이 0이 아닌 주요 평가 예측 계수 측정치 권한을 그럴듯하게 할당 도출 부여받을 치명적인 궤변 모순 위험 요인 가능성이 기저 기형적으로도 엄연히 잠재 도사 존재하고 있기 때문에) 모순 위험 궤도 껍질을 끔찍하게 자극 통제 촉발 배가시켜 증폭 초래하기 때문입니다. 요약해 갈무리하자면 수천수만에 이르는 극도 극한 방대 범위 차수 평가 특성을 손쉽고 만만히 대량 손수 통달 측정 수합 무빙 수집 확보 처리해 도출 대량 쓸어 담아 모을 수 있게 도모 편입해 주는 신기술 발달 최신 무기 도구 패러다임 현장들은 결코 치명적인 엄청난 날카로운 양날의 검과 같음이 입증 보여 집단 확인되는 바입니다. 이것들이 진실로 진정 해결할 산적 손 닿은 코앞 직면한 당장 현장 도출 목표 과제 문제 결론 타겟 종점 객체 실체 무빙과 진짜 밀접하게 핵심 내재 유의 결합 연계된 경우라면 정말 최고로 우수한 황금 찬란 구원 모형 획기적 모델 산출 결과 통달 혁명적 진보 산출 통달 달성을 가져다줄 것이나, 만약 허상 무관 쓸모없는 빈 가짜 객체 껍데기와 불순 무관하게 아무런 쓰레기 무관 관련성이 단절 없는 깡통 소음 찌꺼기가 진득 치장 편입돼 오염된 모의 무대라면 차라리 돌이킬 수 없는 모순 참만 최악의 모델 구제 불능 악화 재앙만 수합 무참한 진영 전복 악영향 패배 나락 결과 붕괴만 필연 참담 토해 산출 몰락 도출 내뱉 멸망 가져오고야 말 것입니다. 심지어 이 요소 변수들이 진짜 100% 결과와 밀접하게 관련이 있는 이상적인 유효 변수 군락일지라도, 모델에 투입해 계수를 세팅하고 적합하는 과정 속에서 필연적으로 초래되어 뒤따르는 분산(variance) 단면의 오답 단위 변동 위험성 몫이 변수 편입으로 얻게 될 편향(bias) 오답의 감소 이점 혜택을 완전히 집어삼키고 능가하여 결과적으로 모델을 갉아먹을지도 모릅니다. “
