# _5.1.1 The Validation Set Approach_ 
# _5.1.1 검증 세트 접근법_

Suppose that we would like to estimate the test error associated with fitting a particular statistical learning method on a set of observations.
하나의 특정 정해진 일련의 관측치 세트 집단 표본들에 대고 특정한 특정 통계적 학습 기법 방법을 타진 적합(fitting) 시키는 행위 타진 양상 과정에 연관 결부 수반 종속 제휴 귀속되는 그 실전 시험 에러율(test error) 을 우리가 유추 연산 추산 견적 계상 가늠 평가 계측 추정해 내고(estimate) 싶어 갈망 염원 열망 도래 열망 타진 희망한다고 일견 가설 상정 가정(Suppose) 해 보라.

The _validation set approach_ , displayed in Figure 5.1, is a very simple strategy validation for this task.
저 앞 Figure 5.1 공간에서 시연 노정 전광 전시 도해 시각 투사 도개 발현 상영 표방 그려져 있다시피 보여지는 바와 공히, 이 _검증 세트 접근법(validation set approach)_ 은 당면한 이 과업 수행 미션 도달 달성 목표 여정 행위 처단 시도 과업에 있어 아주 기초 평이 단조 명료 수월 극히 대단히 무척 통상 평이 단조 평이하게도 매우 대단 극히 극도로 몹시 명료 단순한(simple) 궤적 지표 기치 기점 지표 기전 전략(strategy) 체계 노선 검증책이다.

It involves randomly dividing the available set of observations into two parts, a _training set_ and a _validation set_ or _hold-out set_ .
해당 기법은 기 구축 공수 확보 가용 허용 획득 마련 차용 가능한 여타 잔존 관측치 세트 무리 집단 집합 덩어리들을 여지없이 두 갈래 조각 덩어리 분류 단위 분파 등분 갈래 부분(parts) 들로, 다시 말해 일명 _훈련 세트(training set)_ 조각과 _검증 세트(validation set)_ 혹은 달리 _보류 세트(hold-out set)_ 라 일컫는 두 무리로 완전 임의 무작위 분할 분리 파편 쪼개 무차단 무선 선별 나눔 조각 분리 분할(randomly dividing) 시켜 두는 조작 공정 이행 처단 투여 수행 거동 절차 과정을 기초 필연 수반 통수 포괄 관통 수비 감당 내재 감내 내포 수반(involves) 한다.

The validation model is fit on the training set, and the fitted model is used to predict the responses for the observations in the validation set.
검증 모델 체제 기기 모델은 곧 오직 전수 이격된 저 훈련 세트 장부 위에만 타격 부과 정착 돌림 학습 적용 적합(fit) 구동 처단 되며, 그 후 그렇게 기 훈련 적합을 필한 완료된 해당 핏팅 필 모델 형태 모델 장부는 종국 저 이격 유치 격리 도피 검증 세트 안에 안착 소속 기저 격납 수감된 여타 나머지 대조 관측군 대상 관측치 무리를 대상으로 장차 야기 다가 발생 발발 나타낼 정답 반응 응답 타깃 결괏값(responses) 들을 평가 가늠 계측 조위 계상 산출 타산 예측 추정(predict) 해내는 임무 용처 과제로 기용 투과 동원 도용 투입 쓰임 발동 투입 소모 사용(used) 된다.

The resulting validation set error rate—typically assessed using MSE in the case of a quantitative response—provides an estimate of the test error rate. 
결과 작위 여파 행위 결과 초래 연유 귀결 최종 차제 산물 도달 과정 그 결과 발생 나타난 귀결된(resulting) 그 검증 세트 소 관장 오차 에러율 척도는—보편 대다 평이 통상 전형 보편적으로 정량 연속 지표 양적 정량적 타깃 응답 반응(quantitative response) 척도를 지닌 이수 모델 거거 사안 환경 직면 케이스 경우엔 MSE 공식을 차용 투입 기용 접목 가동 사용 평가 감식 평가 판단 산단 채점 지수 환산 타진 평가(assessed) 도출 받건대—다름 아닌 곧 최종 과업 타깃인 본 시험 오차 에러율의 훌륭 타당 한 가지 타당 추정 치수 가늠 도구 잣대 추정치(estimate) 한 건을 공시 수여 제공 공여 선사 도래 반환 공조 제공(provides) 해준다.

We illustrate the validation set approach on the `Auto` data set.
우리 진영은 저 `Auto` 명칭 데이터 세트 장부를 제단 무대 바탕 삼아 타깃 도구 이 검증 세트 접근 체제 기법의 위력을 시연 설명 도식 전개 주조 입증 조망 해설 묘사 강해 증명 설명(illustrate) 기조 전개할 것이다 보여주려 한다.

Recall from Chapter 3 that there appears to be a non-linear relationship between `mpg` and `horsepower` , and that a model that predicts `mpg` using `horsepower` and `horsepower`[2] gives better results than a model that uses only a linear term.
예전 제 3장 지면 속에서 우리가 애초 `mpg` 요인과 `horsepower` 변수 매개 상호 간에는 다분 일정 곡선 형태 비선형 상관 관계 고리 기조가 맺기 내재 현존 연루 성립 발현 도사 나타나 있음을, 나아가 부연 단순히 일차 단항 단순 선형 항 하나만 고집 투입 반영 사용(uses) 한 직통 라인형 모델 하나보다는 오히려 그 둘의 매개 즉 `horsepower` 와 더불어 덩달 `horsepower`[2] 항목 동시 승차 매개 인자를 이중 모두 동원 사용 결별 가미 수반 이용 기조 사용하여 표적 타깃 `mpg` 수치를 산출 타율 평가 산정 분석 계측 선취 예측 예측(predicts) 도출해 내는 모형 장치 덩치 시스템 모델 결과 모델 쪽이 한층 더 유달 우월 부합 질적 한층 더 우수한 양질 타깃 양립 우수 나은 빼어 빼어난 성적 도달 산출 결과물(better results) 들을 생산 방출 수여 지급 하사 반환 배출 지급 내어 부여(gives) 준다는 진상 팩트 내막 사실 대목을 되려 환기 회고 복기 반환 재고 회상 되새 유추 다시 상고 상기(Recall) 되짚어 보라 떠올려 기억 상기해 두라 보라.

It is natural to wonder whether a cubic or higher-order fit might provide even better results.
여기서 혹여 한술 더 떠 3차 수식(cubic) 규격이나 혹은 그 이상을 넘나드는 고도 초고 수준 고차원 다항 고차(higher-order) 형태 핏 적합 모델을 투과 대입 시도 접목 차용 적용 투입 투여 구동 시도하면 지금보다 훨씬 더 유별 기강 더 나은 훌륭 매서월 한층 빼어난 우수한(even better) 귀결 결과물 산물을 공여 선사 시현 부여 제공 입증 도달 지급 환급 배출 생산 제공(provide) 할 수 있지는 않을까 호기 반문 호기 의아 의문 염문 호기 궁금 질문 기대 의심 의아 자극 착안 망설 의구 의아해하는(wonder) 심리 과정 현상은 무척 합당 지당 필연 자연 정당 순리 타당 지극 응당 필연 자연스런 매한 자연스런 자명 합리 본능 무난 정당 기조 자명 자연스러운(natural) 귀결 이치 도리이다.

We answer this question in Chapter 3 by looking at the p-values associated with a cubic term and higher-order polynomial terms in a linear regression.
우리는 이런 작금 일련 호기 의문 차제 발문 질문 질문(question) 사항에 대고 이미 제 3장 국면 대목 구역 코너 내에서 하나의 선형 회귀 장부 선형 기전 모델 속 이면에 기탁 수반 잔류 동승 얽힌 편제 수록 내재 소속 연관 자리 결탁 자리 잡은 수록된 연관(associated) 되어 기생 중인 그 3차 항 부재와 여타 고위 고차 다항식(higher-order polynomial) 여분 항 잔재 항들에 결부 매달 귀착 결손 붙어 매달 달린 해당 p-값 수위 수치 인계 기율 지표 p-value 지표들을 면밀 꼼꼼 시각 고찰 가늠 조사 관측 주시 고찰 주시 면밀 훑어 조명 관찰 고찰 직면 심층 살펴 타진 살펴봄(looking) 을 무기 요건 해답 근거 삼아 명명 명료 답지 타협 해석 대답 답변 타결 풀이 답변(answer) 해냈다.

But we could also answer this question using the validation method.
허나 지금 당장 우린, 저 복잡 수식을 탈피 대조 대신 이와 대조 지금 방금 채택 습득 배운 검증 방식(validation method) 기법 체제를 응수 도용 투여 기용 동원 대입 차용 전면 활용 수용 응용 동원 차용 사용(using) 해봄으로써도 그 똑같은 동일한 뻔한 일연 공방 호기 발문 동일 질문 궁금증 의문점 의아 의문에 족히 타결 증명 명결 대답 응수 해명 규명 답지 답변 대답(answer) 해명 타결 해소 결론 해소할 응당 지당히 그럴 무난 타진할 여유 공산 능통 수비 그럴 수 있을 그럴 소지가 있다 능통 거뜬할 수 있다.

We randomly split the 392 observations into two sets, a training set containing 196 of the data points, and a validation set containing the remaining 196 observations.
우리는 수록 주어진 392개 정원 관측치 전체 식구 병력 인원 인스턴스 전원을 즉각 무선 공략 마구잡 임의 랜덤 완전 무작위로(randomly) 두 갈래 조각 부락 세트 무리로 확연 공평 나눔 조각 분리 분할 쪼갠다 쪼개 절단 갈라 쪼갠다(split). 즉 196개의 소속 데이터 포인트 멤버 무리를 영입 수납 납치 인볼 내재 귀속 포섭 장착 보유 내포 보유 지닌 포함하는(containing) 훈련 세트 장부 하나와, 이탈 남겨 떨어 나머지 잔존 잔여 생존 남겨 뒤처 잔류 방치 이탈 남겨진(remaining) 여분 196개 수위 인원 관측치 무리를 마저 인계 수용 격납 등극 편입 보유 포괄 담고 안고 품고 내포 보유 지닌 포함하는(containing) 검증 단상 세트 나머지 장부 하나가 바로 그 두 분파 부락 세트들이다.

The validation set error rates that result from fitting various regression models on the training sample and evaluating their performance on the validation sample, using MSE as a measure of validation set error, are shown in the left-hand panel of Figure 5.2.
MSE 오차 수치 적정 공식을 그저 해당 검증 세트 전용 오차 가늠 측도 가치 가늠 측량 단위 저울 척도 기구 지표 척도(measure) 로 삼아 도용 발탁 기용 원용 활용 차용 도용 차용 동원 도입 접목 사용(using) 한 지경 선상 환경 전초 국면에 직면 처해, 훈련 샘플 부락 조각 무리 샘플 대상 위에 대고 여러 다채 상호 각종 가지 다수 수많은 무수 다양한 여러 각양 각양각색 수없이 각기 서로 서로 다른 여러(various) 다채로운 회귀 모델 양태 변종 파생 회귀 장치 기형 기법 모델들을 각기 개별 순차 개별 투입 순차 접목 구동 적용 타격 발동 적합(fitting) 시켜 본 연유 차후 과정 다음 돌려본(from) 후 다시 저 보류된 이탈 격리 보류 유보 검증 샘플 격리 샘플 인원 관측들 그 표본 표면 무리 위에서 타깃 각 모델 변종군 그들의 종국 타율 역량 실력 기량 능률 타율 효율 기능 기능 성적 퍼포먼스 효능 결과 기세 성능(performance) 을 교차 재연 산단 판가 점수 산출 채점 점검 진단 계상 점수 실증 가늠 판측 계측 타진 가늠 판가 채점 평가(evaluating) 해내는 복합 험난 일련 양단 과정 사이클 과정에서 기인 산출 빚어 튀어 도달 도출 창출 발발 발생 터져 유발 초래 잉태 도래 투과 결과 나타 초래 생산 결실 인양 결과로 도출되는(result) 개별 검증 세트 오차 에러율 현황 타율 결손치 무리 덩어리 지분 비율 현황 오차 에러 곡선율들은 아래 Figure 5.2의 좌측 편 패널 단상 무대 도표면 속에 상영 기재 투사 도해 그려 전개 펼쳐 보여 나타 조명 시연 투과 보여 나와 보여(shown) 조명된다.

The validation set MSE for the quadratic fit is considerably smaller than for the linear fit.
해석해 보건대 통찰 2차식(quadratic) 스펙 곡률형 핏 적합 모델 구조 모형이 내뿜 내보 자랑 달성 창출 조달 취득 취득 거둔 검증 세트 MSE 수치 파장 기록 덩치는 그 단순 1차 단일 선형(linear) 핏이 쏘아 수비 내뿜 거스 거시 거둔 챙긴 확보 달성 취득 타율 거둔 기록 수거 취득 취득 거둔 취득한 오차 기록 수치 그것보다 훨씬 상당히 아주 꽤 현저 현격 꽤 대폭 극히 월등 엄청나게 대단히 대폭 상당(considerably) 폭 작게 나은 작게 소소 적게 현저 도달 맺어 더 훌륭 월등 더(smaller) 적다.

However, the validation set MSE for the cubic fit is actually slightly larger than for the quadratic fit.

2. In the validation approach, only a subset of the observations—those that are included in the training set rather than in the validation set—are used to fit the model. Since statistical methods tend to perform worse when trained on fewer observations, this suggests that the validation set error rate may tend to _overestimate_ the test error rate for the model fit on the entire data set. 
둘째(2.), 검증 접근법에서는 오직 전체 관측치의 절반 부분집합(즉 검증 세트 말고 훈련 세트로 뽑힌 녀석들) 만을 튜닝하는 데 사용합니다. 아시다시피 통계 모델들은 훈련하는 데이터 양이 적으면 적을수록 지식이 부족해 그 성능이 현저히 구려지는(perform worse) 태생적 맹점이 있습니다. 이는 결국 무슨 뜻일까요? 절반밖에 훈련을 못 받아 멍청해진 이 모델이 내뱉는 검증 세트 오차율은, 실제 전체 데이터를 다 먹고 똑똑하게 훈련했을 최종 온전한 모델의 진짜 테스트 오차율보다 본의 아니게 성능 문제를 너무 엉망으로 깎아내려 **에러가 더 심각한 것처럼 과대평가(overestimate)** 하는 가혹한 편향 왜곡 부작용을 낳고 만다는 치명타를 시사합니다.

In the coming subsections, we will present _cross-validation_ , a refinement of the validation set approach that addresses these two issues. 
그래서 다가오는 다음 하위 섹션들에서는, 당장 직면한 이 반반 분할의 치명적인 두 가지 병폐(관측치 낭비와 널뛰기 결과) 를 아주 스마트하게 타파 해결해 내는 검증 세트 접근법의 최종 진화형 궁극기, 즉 **교차 검증(Cross-Validation, CV)** 이라는 위대한 구원 기술을 본격적으로 등판시켜 소개해 드리겠습니다!
