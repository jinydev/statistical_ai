# Acknowledgements

# 감사의 글

A few of the plots in this book were taken from ESL: Figures 6.7, 8.3, and 12.14.

이 귀중한 책의 출간에 기여한 도서 본문 속의 아주 극히 그 일부분에 해당하는 몇몇 소수의 여러 통계적 도해 모형 그림 그래프(plots)들의 경우, 본서의 영감에 지대한 기여를 한 ESL 서적으로부터 몇몇 일부분을 인용하고 차용하여 고스란히 발췌해서 가져왔습니다. 대상 인용 그림은 구체적으로 다음 항목입니다: 그림(Figures) 6.7, 8.3 그리고 12.14 내용.

All other plots were produced for the R version of ISL, except for Figure 13.10 which differs because of the Python software supporting the plot.

본 파이썬 지원 판본에 있어서, 해당 내용을 온전히 유지하게끔 뒷받침하고 지원해 내는 핵심적인 Python 기반의 주된 소프트웨어적 여러 요건과 내부 구조의 차이점으로 기인하여 약간의 궤를 달리하며 상이하게 차이가 발생하는 본문 그림 13.10의 아주 극히 드문 사례에 해당하는 일부분 케이스를 유일한 단 하나의 특정 예외 조항으로 따로 분리하고 제외하고서, 거의 예외 없는 다른 부류의 기타 모든 대다수 나머지 다른 모든 전체적인 차트 모양과 종류의 도해 모형 그림들은 당초 원 구상 기획이었던 과거 시절에 집필 구상된 애초에 구 ISL의 근본 R 환경 프로그래밍 언어의 ISL R 버전용으로 과거 제작되었던 고유한 원본을 사용하여 본 도서에 수록, 제작하고 생산하여 기입 후 반영해 산출했습니다.
