# 감사의 글

A few of the plots in this book were taken from ESL: Figures 6.7, 8.3, and 12.14.

이 책의 플롯 중 일부인 그림 6.7, 8.3 및 12.14는 ESL에서 가져왔습니다.

All other plots were produced for the R version of ISL, except for Figure 13.10 which differs because of the Python software supporting the plot.

그림 13.10의 경우 플롯을 지원하는 Python 소프트웨어로 인해 서로 다르다는 예외 사항을 제외하면, 기타 다른 모든 플롯은 ISL의 원 텍스트인 R 버전을 위해 렌더링 제작되었습니다.
