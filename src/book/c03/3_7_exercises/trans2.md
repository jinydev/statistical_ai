# 3.7 Exercises

---


### Conceptual (개념 문제)

데이터셋이 없는 상황에서, 잔차 제곱합을 직접 수식으로 풀거나 특정 가설 하에서 회귀 계수의 성질을 수학적으로 풀어보는 수학 올림피아드 이론 훈련 세트입니다. (머리가 조금 아플 수 있어요!)
다중 회귀 뼈대 및 가설 검정 판사봉의 원리에 대한 아주 깊고 탄탄한 통계적 기초를 다잡습니다.

### Applied (응용 문제)

이론은 끝났다! 파이썬을 실전처럼 켜고 교재의 오픈 데이터(예: 앞서 배운 카시트 데이터 등)로 회귀 분석을 직접 땀 흘리며 돌리고, 잔차 진단 플롯 그림을 그려보며 모델의 문제점을 스스로 캐내 보고하는 고강도 실습 테스트입니다!
여러분을 현업 실무 연봉 1억 분석가로 이끌어 줄 실전 근육 감각을 양성합니다.
