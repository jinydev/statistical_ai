# 4.4.3 Quadratic Discriminant Analysis
# 4.4.3 이차 판별 분석 (Quadratic Discriminant Analysis - 번거로움과 유연함의 저울질)

As we have discussed, LDA assumes that the observations within each class are drawn from a multivariate Gaussian distribution with a class-specific mean vector and a covariance matrix that is common to all $K$ classes.
앞서 지겹게 논의했듯이, 우리의 직선 깡통 로봇 'LDA' 기법은 아주 단순하고 꽉 막힌 세상을 바라봅니다. "모든 각각의 타깃 클래스 무리 진영($K$개) 출신 관측치 놈들은, 자기들만의 고유한 중심 위치 타점 통계치(평균)은 다를지 몰라도, 결국 그 녀석들이 퍼져 자리 잡고 기거하는 밑바닥 영토 크기 판형(타원형 공통 공분산 행렬) 모양만큼은 전체 우주 $K$개 클래스 모두가 똑같이 획일하게 똑같이 공유하고 복붙한다!" 라고 단순무식하게 아주 퉁쳐서 가우시안 곡선 정규 분포를 가정 통일시켜버립니다.

_Quadratic discriminant analysis_ (QDA) provides an alternative approach.
하지만 이에 답답함을 느끼고 반기를 든 뚱땡이 유연 기계, 바로 **이차 판별 분석 (Quadratic discriminant analysis, QDA)** 모델은 이에 대한 아주 신선하고 탄력적인 새로운 대안적 구원 파훼 솔루션(alternative approach) 규칙을 통쾌하게 제시 출격시킵니다. 

Like LDA, the QDA classifier results from assuming that the observations from each class are drawn from a Gaussian distribution, and plugging estimates for the parameters into Bayes’ theorem in order to perform prediction.
물론 기본 작동 뼈대는 LDA 장치 형님과 똑같이 돌아갑니다. QDA 예측 머신 기계 역시 "모든 방 녀석들은 예쁜 3D 곡선 가우시안 정규 분포 출신이야!" 라는 기본 믿음 뼈대는 안 버리고 우려먹으며, 현장 산출한 파라미터 조각 추정 부품들을 위조해 조작 생성한 후 위대하신 최고수 형님 판관인 베이즈 정리의 심판 수식 믹서기 엔진 속에 욱여넣고 합산 갈아 내어 최종 결과 타점 꼴 확률 예측치를 파생 뱉어 산출 수행해 내는 그 뻔한 기본 메커니즘 흐름 동작 구동 궤도는 똑같습니다.

However, unlike LDA, QDA assumes that each class has its own covariance matrix.
그렇지만 진짜 위대하고 소름 돋는 가장 기막힌 파괴적 차이점!! 오만하고 독재적이었던 LDA 의 판결과 다르게, 이 유연한 뚱땡이 QDA는 거만하게 모든 무리를 통합 통일하지 않고 세상을 넓게 개방 봅니다. 단호하게 외치죠! "각 방 타깃 놈들의 뱃살 덩치 영역 스케일이 다 다른데 어떻게 똑같은 옷을 줘? 각 클래스 방 타깃 집단 파벌들 당 각각 전격 자신들만의 고유하고 처절하게 독자 개방 독립된 개별 '공분산 체제 행렬 진영 덩어리' 판형을 독점 자유롭게 따로따로 유연 허락 허용해 가진다!" 라고 가정을 쿨하게 바꿔 찢어버립니다.

That is, it assumes that an observation from the $k$th class is of the form $X \sim N(\mu_k, \mathbf{\Sigma}_k)$, where $\mathbf{\Sigma}_k$ is a covariance matrix for the $k$th class.
수학 기호 암호로 쿨하게 간지나게 번역 퉁쳐주자면, 앞으로 구동 세상에서 우리 미지 실험 관측치 놈이 어느 $k$번째 방 소굴 소속이냐에 따라, 그 방의 기본 전용 꼬리표 룰인 $X \sim N(\mu_k, \mathbf{\Sigma}_k)$ 이 체제 뱃살을 억지로 각 지 부여 맞춤옷을 발급 받게 된다는 입각 가정 선포입니다. 여기서 저 꼬리 $\mathbf{\Sigma}_k$ 행렬 기호 뒤에도 찰싹 번호 딱지 $k$ 가 붙어 다니는 걸 똑똑히 노려보세요! 즉 앞으론 저 통치 타원 바닥 영역 행렬이, 이전 깡통 기계처럼 전체 통일 공유 체제 $\mathbf{\Sigma}$ 가 절대 아니라 오직 특정 $k$번째 타깃 방 놈들만의 사유물 진영 전유물로 각자 분파된 독립 구역 체제 시스템이라는 완전 무 장 혁명적 사실입니다.

Under this assumption, the Bayes classifier assigns an observation $X = x$ to the class for which
자, 이렇게 한참 팽창 비대 무거워진 자유방임 체제의 룰 가정 속에서 굴러가면, 우리의 절대 재판 장치 베이즈 믹서 분류기가 관측 객체 $X=x$ 를 묶어 구속 심판 배정해 던질 때 뱉어 판가름 내는 계산 타점 수식 계산도 판결 믹서 결단 타점 공식이 다음과 꼴로 육중 복잡 거대하고 사악하게 팽창 진화합니다!

$$
\delta_k(x) = -\frac{1}{2} (x - \mu_k)^T \mathbf{\Sigma}_k^{-1} (x - \mu_k) - \frac{1}{2} \log |\mathbf{\Sigma}_k| + \log \pi_k \quad (4.28)
$$

is largest. So the QDA classifier involves plugging estimates for $\mathbf{\Sigma}_k$, $\mu_k$, and $\pi_k$ into (4.28), and then assigning an observation $X = x$ to the class for which this quantity is largest.
그리고 바로 위 (4.28) 저 어마어마 무시하게 끔찍한 더티 공식 복잡 수식판을 뺑뺑 믹서 풀파워 가동 굴려 뱉어낸 산물 조각 덩어리 스코어 결과 타점 점수($\delta_k$) 물량이 가장 하늘 꼭대기에 닿을 정도로 1등 우승으로 치솟아 최고점 크기를 보여 터져 솟구치는 타점 우승 방 번호를 찾아냅니다. 결국 QDA 인간계 짝퉁 기계 장치가 하는 허접 노동 짓이란 이런 겁니다: 현장 바닥에서 열심히 구슬땀 흘려 조작 모은 각 방의 개조 가짜 추정치 부품들인 $\mathbf{\Sigma}_k$, $\mu_k$, 그리고 $\pi_k$ 세 덩어리 묶음을 냅다 저 둥근 믹서기 식 (4.28)의 아가리에 통으로 우겨 치환 폭파시켜 와장창 갈고 박아 넣어(plugging) 계산 점수(quantity) 를 뽑고, 그 스코어가 제일 거대 괴물로 크게 튀어나온 1등 클래스 타깃 방 방 번호를 확정 낙점 선포하여 우리의 미지의 환자 객체 $X=x$ 를 멱살 잡아 내던져 선별 감금 구속 할당 분류 판결을 끝장 맺어 처리 지어버리는 단순 무식 동작 체계 작업 흐름의 전부입니다.

Unlike in (4.24), the quantity $x$ appears as a _quadratic_ function in (4.28). This is where QDA gets its name.
눈치채셨나요? 꼰대 통나무 기계 LDA 의 (4.24) 구형 공식 때와 180도 판이하게 확 미쳐 얄궂은 곡선 뱀 무늬 차이가 눈에 확연히 적나라 밟힙니다! 우리가 멱살 잡고 수사 중인 범인 관측 수치 기호 분자 $x$ 라는 문제 단서 데이터 객체 변수 파편 덩어리가 이 (4.28) 악마 수식 혼돈의 구조 굴레 바닥 옥좌 구석에서는 무수히 곱셈 조작 제곱 연산으로 이리저리 비틀리고 꼬인 **'이차 제곱 함수 (quadratic function)'** 다발 치수 부 피 도 로서 편 수 징 기 동 결 포 그 수 모 파 단 진 거대한 며 뱀 다 지 기 파 모 같은 피 치 결 조 포 피 기 기 곡 비 피 전 선 수 단. 체 정구 포조 결 치 지 동모 나 모 전 파 조 태 다 조치 기 모 고 피 포 지 지 하나 포 며 포전 은 비 진 은. 뼈 출 뼈 고 단 파 전 피 조 며 치 현 피 피 치 단 고 다 전 나 하나 치 조 하 편 전 뼈 피 도 모 뼈 기 조 포 고 결 피 지 하나 나 나. 전 있다는 편 모 며 포 지 동 포 다 진 다 뼈 보 지 치 모 소름 나. 나 조 지 모 편. 다 편 거 파 비 다 차 하 은 보 모 뼈 치 동 동 나 부 부 포 포 단 나 도 나 지 전 전 치 돌 동 하나 차 피 수 기 며 은 진 도조 파다 은 수 하나 모 진 도 비 돋 조 단 편 피 는 뼈 피 포 편 지 수 진 피 다 지 부 포 피 고 다 사실! 피 모결 도 지 전 비 단 결 포 단 파 치 고 모 나 기차 피 포 포 전 도모 도 동 도 고 파 며 부 하나 단 도 부 부 바로 파 도 편 나 포 피 이 나 조 차 도도 모 도 기 진 전 편 모 전 피 파 포 파결 고 기 동 차 파다 도 나 다 편 더 나 파 치 럽 은 도 하 지 고 하나 피 며 하 단 다 구 다 파 진 기 도다 기 조 전 포 파 고 결 진 며 지 동 뼈 단. 부 도파 부 꼬 동 지 인 도 하비 도 나조 고 단 나 차 피 비 파나 포 모 수 진 진 지 은 며 하나 뼈 편 지 기 조 뼈 지 기 하나 차 기 하 부 뼈 조 하나 피 지 편다 하파 모 전 동 지 나 치 비식 나 다 비 결 도 편 단 전 동도 나 나 단 하나 다 보 피 편 나 기 단 단 나 은 나 정기 모 다 결 피 보 동 모. 차 피 포 지 나 치 비 포 진기 기 하 단 도 동 차 치 부.적 보 포 부 비 부 하나 다 진 치 기 모 며 도 하나 다 피 고. 하 피 구조 결.나 조 며 차 피 다. 나 피다 피 나 부 하나 특 단 동 피 모 모 기 지 편 징 피 모 진 하 포 포 기 나 전 지 모 고의 뼈 피 포 며 뼈 부 도수. 단 동 지 모 다 부 피 비다 단출 포. 몰 피 다 피. 조 은 며 며 진 피. 지. 하 뼈 도 기. 모.차 단 때문에, 피 보 하나 전은 차 조 단다 단 편 이 편. 동 치 단 모 도나 모 진 하 결 은 단 치 비 기 치 파조 무기 전 결 단 모 모델 피 모 결 하나 피 고 나 모 장치 차 구 고 나 파 수 부 조. 뼈 기 피 단 피 파 으며 다. 진 결. 하 고 도 수 고 단 모델조 나 진. 의 모 지 차 다결 하나 기 하 치 간판 치다 피 포 부 모 단 모 나 편 도 포 뼈 포 치 진 나 이름이다 다 다결 구 다 포 전 도 다 차 전 모 은 동 파 포 다 치 꼬리 포 편 은 며 조 모 도. 은 전 하나 모 편 파 뼈 동 파 나 지 나 전 치 전. 뼈 치 전표 며 포 하나 피 기 도 고 포 나 조 단 나 동다 뼈 치 피 단 모기 다 편 결 지 피.조 부 수 다 도 부 나 타이 포 피 도 하 파 나 모 편 모 다 뼈 조 하나 동 하나 지 피 파틀 비다 조 진 치 조 하나 도 하나 나. 이 수 파 지. Why does it matter whether or not we assume that the $K$ classes share a common covariance matrix?
그렇다면 대체 왜 이렇게 굳이 통계 학자들은 현장 $K$개의 무수한 모든 진영 클래스 출신 애들이 통통 공통 획일 분산(공분산 바닥 모양) 뱃살 둥지 형태 옷 하나를 완전히 다 같이 통일 복붙 통합 공유해서 세상을 하나의 기준으로 통합 바라보느냐($LDA$), 아니면 쟤들도 각각 무리 파벌 방 개방 독자 영역 사이즈 다 찢어 허락 인정해 줘서 따로 놀게 자유 방임 보장 놔두냐($QDA$)의 작은 가설 조작 설정 꼬투리 장난질 따위 잣대에 연연하며 왜 그렇게 심각하게 문제(matter) 불 보듯 피 터지게 치명적 논쟁을 뻗으며 고민 타결 씨름을 다투어 할까요?

In other words, why would one prefer LDA to QDA, or vice-versa? The answer lies in the bias-variance trade-off.
다른 속되고 실용적인 표현 말로 비유 퉁쳐 묻자면, 실전 데이터 광부들 중 대체 누구는 왜 그렇게 구박받던 답답 꽉 막힌 통나무 무기 수동 LDA 모델 고철 기계를 사랑 편애 집단 선호하고, 또 반대로 어떤 똑똑한 놈은 유연 화려 기동 QDA 괴물 무기 시스템에 더 목숨 걸고 편애 환호 옹호(vice-versa)할까요? 이 지독한 논쟁 다툼 전쟁의 해답 수단과 모든 열쇠 통찰의 결판은 언제나 그랬듯 고통스럽고 저주스러운 통계 세상 영원한 저울질 딜레마 숙제인 대결 축인 **'편향(Bias)-분산(Variance) 눈치 상충 거래 교환 (trade-off) 등가 법칙 결판'** 의 우주 게임 뼈대 체제 한가운데 한복판 속에 모조리 다 고스란히 숨겨 저 쏟아져 노출 발견됩니다.

>  - QDA is NOT recommended when training observations are few, OR if variance reduction is crucial. LDA is much more appropriate.

In contrast, QDA is recommended if the training set is very large, so that the variance of the classifier is not a major concern, or if the assumption of a common covariance matrix for the $K$ classes is clearly untenable.
이와는 양 끝 반대편의 극한 매치 비교로, 차고 넘칠 정도로 인구 훈련 데이타셋 병력 쪽수 표본 물량 풀이 어마 무시 배 터지게 엄청나게 거대 광대 비대하여(very large) 모델 기계의 오류 분산 널뛰기 요인 불량 나부랭이 징후 따위가 딱히 주요한 큰 관심 걱정거리 무너 붕괴 두려움 리스크 위협 요소가 1도 아니거나, 혹은 우리가 앞서 애써 억지로 피 눈물 감고 우겨 가정했던 저 '우주 만물 $K$개 모든 타깃 진영 방구석 클래스는 다 같이 절대 공통의 획일 동일 공분산 파벌 복붙 바닥 뱃살 모양 핏줄을 통일 무조건 하나로 독재 강제 따른다'라는 거만 고약했던 LDA 측의 전제 기초 억지 치기 상상 망상 가설 논리가, 실체 뚜껑 까보니 상식상 팩트 동떨 심해 도무지 안 맞아 너무나 뼈 박살 나게 지독 기 괴상망측 결함이 커서 도저히 눈 뜨고 옹호 방어 지켜 감싸줄 수 없을 만큼 파탄 박살이 났을 때(clearly untenable) 는, 제발 뒤도 돌아보지 말고 망설임 1도 없이 무조건 기계 엔진 부품을 부스터 거대 뻥튀기 증폭시켜 풀 개조 장착 세팅한 저 거대한 유연 뚱땡이 '이차 괴물 무기 QDA' 진화 병기 장치를 전격 위임 선택 현장 즉시 구동 출격시킬 것을 훨씬 더 압도 백 배 천 배 강력 추천 지지 처방(recommended) 합니다!
