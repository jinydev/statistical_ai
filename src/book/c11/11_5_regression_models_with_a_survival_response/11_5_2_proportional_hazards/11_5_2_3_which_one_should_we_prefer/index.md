# Which one should we prefer? 

In fact, there is a close relationship between these two approaches. In particular, when taking Approach #1, there are a number of possible ways to test _H_ 0. One way is known as a score test. It turns out that in the case of a single binary covariate, the score test for _H_ 0 : _β_ = 0 in Cox’s proportional hazards model is exactly equal to the log-rank test. In other words, it does not matter whether we take Approach #1 or Approach #2!
