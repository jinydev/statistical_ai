# 6.3.2 부분 최소 제곱법 (Partial Least Squares)

The PCR approach that we just described involves identifying linear combinations, or _directions_, that best represent the predictors $X_1, \ldots, X_p$. These directions are identified in an _unsupervised_ way, since the response $Y$ is not used to help determine the principal component directions. That is, the response does not _supervise_ the identification of the principal components. Consequently, PCR suffers from a drawback: there is no guarantee 
우리가 방금 살펴본 PCR(주성분 회귀) 방식은 특징 변수 $X_1, \ldots, X_p$의 분산을 최대한 잘 보존하는 선형 축, 다시 말해 _방향(directions)_ 을 식별하는 접근법입니다. 그런데 이런 방향 축들은 완전히 _비지도(unsupervised)_ 학습 방식으로 추출됩니다. 왜냐하면 주성분의 궤적을 찾을 때 최종 타겟인 반응 변수 $Y$ 데이터가 '컨닝 페이퍼'처럼 활용되지 않고 철저히 배제되기 때문입니다. 즉, 반응 변수는 주성분들을 솎아내는 과정에 있어 그 어떤 개입이나 _지도(supervise)_ 도 하지 않습니다. 이로 인해 PCR은 치명적인 한 가지 단점을 끌어안게 됩니다.

We now present _partial least squares_ (PLS), a _supervised_ alternative to PCR. Like PCR, PLS is a dimension reduction method, which first identifies a new set of features $Z_1, \ldots, Z_M$ that are linear combinations of the original features, and then fits a linear model via least squares using these $M$ new features. But unlike PCR, PLS identifies these new features in a supervised way—that is, it makes use of the response $Y$ in order to identify new features that not only approximate the old features well, but also that _are related to the response_. Roughly speaking, the PLS approach attempts to find directions that help explain both the response and the predictors. 
자, 그래서 이제 우리는 그 아쉬운 PCR 궤적을 대체할 훌륭한 _지도적(supervised)_ 심화 대안 모델인 _부분 최소 제곱법(partial least squares, PLS)_ 구조를 본격적으로 소개합니다. PCR 방식과 마찬가지로, PLS 역시 일종의 차원 축소 모형입니다. 초창기 오리지널 특성들을 선형으로 응축 융합해낸 $Z_1, \ldots, Z_M$ 이라는 신규 피처(특성) 세트를 솎아낸 다음, 이 한정된 $M$개의 파생 축들을 활용해 최소 제곱법으로 선형 모델을 적합시키는 방식은 같습니다. 하지만 PCR 기법과 결정적으로 다르게, PLS는 새로 마련된 이 특성 다발들을 이른바 "지도적"인 방식에 의거해 차출해냅니다—다시 말해, 단순히 예전 특성들을 엇비슷이 잘 품어내는 것을 넘어서 아예 애초 타겟인 _반응 변수 정답지에 실질적으로 밀접하게 맞물려 관련있는(related to the response)_ 파생 피처를 찾고자 반응 변수 $Y$를 적극적으로 개입시켜 활용합니다. 대략적으로 말하자면, PLS 수단은 반응 변수와 입력 예측 변수 양쪽 모두를 동시에 훌륭히 잘 설명해 주는 궁극의 타협점 차원 방향선을 모색하려는 시도라고 할 수 있습니다.

We now describe how the first PLS direction is computed. After standardizing the $p$ predictors, PLS computes the first direction $Z_1$ by setting each $\phi_{j1}$ in (6.16) equal to the coefficient from the simple linear regression of $Y$ onto $X_j$. One can show that this coefficient is proportional to the correlation between $Y$ and $X_j$. Hence, in computing $Z_1 = \sum_{j=1}^p \phi_{j1} X_j$, PLS places the highest weight on the variables that are most strongly related to the response. 
이제 이 PLS의 첫 1순위 위상 방향선이 어떻게 도출되는지 살펴보겠습니다. 준비 단계로 모든 $p$개의 예측 변수를 각각 표준화(standardizing)하고 난 후, PLS 시스템은 반응 타겟값 $Y$를 개별 단일 $X_j$ 요소에 맞춰 단순 선형 회귀시켰을 때 나오는 그 계수값 수량과 식 (6.16)의 개별 할당 수치 $\phi_{j1}$ 파라미터를 그대로 복사 붙여넣기 하듯 동일하게 배정시켜 제 1 위상의 주성분 축 $Z_1$을 계산 창출해냅니다. 우리가 여기서 수학적으로 수식 증명해 낼 수 있는 재밌는 현상은, 바로 이 단순 회귀 계수 수치가 'Y 반응치와 개별 $X_j$ 타겟 수치 사이 공간 피어슨 상관계수(correlation)' 크기 지수에 정비례하여 동조한다는 점입니다. 이 원리에 의거하여 $Z_1 = \sum_{j=1}^p \phi_{j1} X_j$ 기둥 편제 축을 도출할 때 조작 속성을 뜯어보면, PLS는 자연스럽게 기둥 모델 타겟 목표인 반응 변수에 가장 직결되어 강하게 짝을 지은 유의미 변수 녀석들에게 제일 가중 무거운 엄청난 영향력 가중치(highest weight) 패널티 배율 척도를 우선적으로 부과하게 지령을 두게 되는 셈입니다.

Figure 6.21 displays an example of PLS on a synthetic dataset with Sales in each of 100 regions as the response, and two predictors; Population Size and Advertising Spending. The solid green line indicates the first PLS direction, while the dotted line shows the first principal component direction. PLS has chosen a direction that has less change in the `ad` dimension per unit change in the `pop` dimension, relative to PCA. This suggests that `pop` is more highly correlated with the response than is `ad`. The PLS direction does not fit the predictors as closely as does PCA, but it does a better job explaining the response. 
그림 6.21 도면은 각 100개의 다른 지역 판매량 `Sales` 수치를 목표 반응 타겟 변수로 선정하고 덧붙여 '인구 규모(Population Size)'와 '광고 지출(Advertising Spending)'이라는 딱 두 가지 예측 변수를 쌍으로 장착시킨 가상 장치 합성 데이터셋 모델상에, PLS 측정 시스템 진영을 적용해 구동시켜 본 도출 예시 화면입니다. 진하게 그려진 단색 실선 궤도는 첫 번째 위상의 대표 PLS 단면 방향축을 나타내는 반면, 무단 점선 마디는 대조군인 최선두 제 1 주성분 추출의 PCA 측정 방향 구도를 동시에 병행 비교해 보이고 있습니다. 여기서 PLS의 튜닝 선택 방향은 투박한 기성 PCA 방식 모델 산출에 비해 비교해 따져 보면, 통계 단면 `pop` 차원이 1 단위(unit) 상승 변화할 때마다 맞물려 나타나는 `ad` 차단 지표 면의 반동 폭이 더 미세히 적게 나타나는 쪽으로 그 굴절 방향 좌표를 수렴시켰습니다. 이러한 산출 조율 판단은, `pop` 지표라는 인자가 타 지표 자원인 `ad` 자원에 대비해 비교도 안 될 만큼 결과 반응치에 단단히 강인하게 종속 직결되어 훨씬 더 월등하고 지독하게 높은 수위의 상관 연관성(highly correlated)을 굳건히 확보 공유하고 있다는 근거 단서를 수리 지시 시사합니다. 반면에 비록 PLS 방향선이 PCA 방식만큼 기존의 입력 예측 변수들을 타이트하게 다 껴안아 밀착 설명(fit)해주진 못하지만, 궁극적으로 목표 타겟 반응 변수를 예측해 내고 설명해 주는 역무(better job)에 있어서만큼은 확연히 더 나은 우수한 성능 결과를 보여 줍니다.

To identify the second PLS direction we first _adjust_ each of the variables for $Z_1$, by regressing each variable on $Z_1$ and taking _residuals_. These residuals can be interpreted as the remaining information that has not been explained by the first PLS direction. We then compute $Z_2$ using this _orthogonalized_ data in exactly the same fashion as $Z_1$ was computed based on the original data. This iterative approach can be repeated $M$ times to identify multiple PLS components $Z_1, \ldots, Z_M$. Finally, at the end of this procedure, we use least squares to fit a linear model to predict $Y$ using $Z_1, \ldots, Z_M$ in exactly the same fashion as for PCR. 
두 번째 차례의 PLS 방향 좌표를 찾아내기 위해, 우리는 일단 각 예측 변수들을 이미 확보된 선두 척도 $Z_1$ 축에 맞춰 선형 회귀시킵니다. 그렇게 얻어진 _잔차(residuals)_ 덩어리들을 활용함으로써 각 변수들을 $Z_1$ 에 관하여 _조절(adjust)_ 하는 일련의 튜닝 스텝을 먼저 밞습니다. 이 계산된 잔차 덩어리들은 첫 번째 1순위 PLS 방향 지표만으로는 미처 다 해명되지 못하고 통계 정보망에 고스란히 남아 있는 나머지 잔여 정보(remaining information) 데이터 요소들로 해석될 수 있습니다. 이렇게 _직교화 분할 과정을 거친(orthogonalized)_ 정제 데이터를 새로운 훈련장으로 활용하여, 최초 생 초창기 데이터들을 바탕으로 첫 주성분 $Z_1$ 객체를 계산해 냈을 때와 기하학적으로 전면 똑같은 방식 궤적(exactly the same fashion) 요령 그대로를 다시 반복해 두 번째 주성분 개체인 $Z_2$ 축을 이어서 산출 계산해 냅니다. 이렇듯 순환 점진적으로 꼬리를 무는 다람쥐 쳇바퀴 식의 접근법 궤도는 우리가 원하는 요구 한계 목표치인 총 $M$ 번까지 연속 반복 동원될 수 있으며, 그 기나긴 절차의 끝에 다다르면 다차원 구조의 무수한 PLS 주성분 다발인 $Z_1, \ldots, Z_M$ 편제 객체 무리 전체를 모조리 모아 수집 식별해 낼 수 있게 됩니다. 마침내 이 파생 과정들의 막바지 마지막 마감 국면에 도달하게 되면, 우리는 최종 목표인 오리지널 반응 종속 타겟 $Y$ 척도를 예측해 내려는 단선 선형 회귀 적합 모델 진영 엔진에다 앞서 거둔 저 파생 수확물 $Z_1, \ldots, Z_M$ 요소들만을 투입 활용합니다. 이때는 과거 PCR 모델 평가를 위해 실시했던 궤적 기구와 머리끝부터 발끝까지 전면 동등 일치 구결하는 과정 궤도(in exactly the same fashion as for PCR) 매커니즘 정석 그대로를 대입하여 최소 제곱법 기반의 선형 모델 적합을 시원하게 완료 수행하게 됩니다.

As with PCR, the number $M$ of partial least squares directions used in PLS is a tuning parameter that is typically chosen by cross-validation. We generally standardize the predictors and response before performing PLS. 
고전 PCR 조처가 그러했듯, 실무 PLS 파이프라인 무대 진영 안에 최종적으로 취합되어 쓰이게 될 최소 부분 제곱 방향선 허용 개수 척도인 $M$ 지수는 일종의 시스템 조율 설정 통제 지표인 튜닝 파라미터(tuning parameter)입니다. 이 숫자는 관행적으로 통계학에서 가장 믿음직하고 널리 쓰이는 표본 잣대인 **교차 검증(cross-validation)** 평가 테스트 산출 성적표 체제에 전적으로 의거해 평가되고 최종 선택 책정됩니다. 한 가지 실무 필수 팁으로, 모델 훈련가는 실제 PLS 파이프라인 예측 기동 수행 단계 착수 개시 바로 직전 기초 단계에서, 수많은 무수한 모든 초기 입력 예측 변수 자원 무리들과 그에 맞서는 짝꿍 타겟 측 반응 응답 변수 양쪽 진영 모두를 의무적으로 몽땅 일괄 **표준화(standardize)** 시키는 평준 작업 비율 튜닝 제재 조치를 가장 일반적으로 무조건 선 적용해 밟아주는 것이 국룰 원칙입니다.

PLS is popular in the field of chemometrics, where many variables arise from digitized spectrometry signals. In practice it often performs no better than ridge regression or PCR. While the supervised dimension reduction of PLS can reduce bias, it also has the potential to increase variance, so that the overall benefit of PLS relative to PCR is a wash.
