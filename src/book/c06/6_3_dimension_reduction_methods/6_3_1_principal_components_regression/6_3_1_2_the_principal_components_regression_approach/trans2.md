# 6.3.1.2 주성분 회귀 접근법 (The Principal Components Regression Approach)

The _principal components regression_ (PCR) approach involves constructing the first $M$ principal components, $Z_1, \ldots, Z_M$, and then using these components as the predictors in a linear regression model that is fit using least squares. The key idea is that often a small number of principal components suffice to explain most of the variability in the data, as well as the relationship with the response. In other words, we assume that _the directions in which $X_1, \ldots, X_p$ show the most variation are the directions that are associated with $Y$_. While this assumption is not guaranteed to be true, it often turns out to be a reasonable enough approximation to give good results. 
**주성분 회귀(principal components regression, PCR)** 접근법은 앞에서 구한 $M$개의 주성분 $Z_1, \ldots, Z_M$을 새로운 예측 변수로 사용하여 최소 제곱법 기반 선형 회귀 모델에 적합시키는 기법입니다. 이 기법의 핵심적인 아이디어는 보통 "소수의 몇 개 주성분"만으로도 데이터가 지닌 전체 분산의 대부분을 훌륭히 설명할 수 있고, 반응 변수 $Y$와의 관계성까지 충분히 설명해 낸다는 것에 있습니다. 즉, 우리는 _$X_1, \ldots, X_p$ 원본 변수들이 공간상에서 가장 크게 변동하는 방향이 곧 반응 변수 $Y$와 밀접하게 연관된 방향일 것이다_ 라고 가정합니다. 항상 옳은 것은 아니지만, 실전에서는 좋은 결과를 주는 합리적인 근사법으로 자주 통용됩니다.

If the assumption underlying PCR holds, then fitting a least squares model to $Z_1, \ldots, Z_M$ will lead to better results than fitting a least squares model to $X_1, \ldots, X_p$, since most or all of the information in the data that relates to the response is contained in $Z_1, \ldots, Z_M$, and by estimating only $M \ll p$ coefficients we can mitigate overfitting. In the advertising data, the first principal component explains most of the variance in both `pop` and `ad`, so a principal component regression that uses this single variable to predict some response of interest, such as `sales`, will likely perform quite well. 
만약 PCR의 핵심 가정이 맞아떨어진다면, $Z_1, \ldots, Z_M$ 성분들에게 최소 제곱법 회귀를 적합시키는 것이 모든 원본 변수 $X_1, \ldots, X_p$ 를 다 쓸 때보다 훨씬 나은 예측 성과를 가져다 줍니다. 타겟 $Y$ 변수를 파악하는 데 필요한 정보 대부분이 이미 소수의 $Z_1, \ldots, Z_M$ 에 압축되어 있기 때문입니다. 게다가 추정해야 할 계수도 $p$개에서 훨씬 적은 $M$개로 줄어들므로, 과적합(overfitting) 문제마저 완화시킬 수 있습니다. 앞서 본 광고 데이터 예시에서, 제 1 주성분 한 개가 투입 지표 `pop`과 `ad` 양쪽의 분산을 잘 대변했습니다. 따라서 이것 단 하나만 변수로 써서 `sales` (매출) 같은 목표를 예측하는 주성분 회귀 모델은 아주 탁월하게 작동할 가능성이 큽니다.

The relatively worse performance of PCR in Figure 6.18 is a consequence of the fact that the data were generated in such a way that many principal components are required in order to adequately model the response. In contrast, PCR will tend to do well in cases when the first few principal components are sufficient to capture most of the variation in the predictors as well as the relationship with the response. The left-hand panel of Figure 6.19 illustrates the results from another simulated data set designed to be more favorable to PCR. Here the response was generated in such a way that it depends exclusively on the first five principal components. Now the bias drops to zero rapidly as $M$, the number of principal components used in PCR, increases. The mean squared error displays a clear minimum at $M = 5$. The right-hand panel of Figure 6.19 displays the results on these data using ridge regression and the lasso. All three methods offer a significant improvement over least squares. However, PCR and ridge regression slightly outperform the lasso.

이 무대 환경 안에서 이제 투입 운용 주성분 누적 단위 척도 지수 숫자인 모형 한계 $M$ 이 늘어남에 발맞춰, 그토록 내재 요동치던 오차 편향 착각 요인(bias)이 신기루처럼 급전 순식간 하락 절벽 곤두박질 계단으로 이어져 영점 원좌표 지점 $0$(zero)을 겨냥 향해 막힘없이 돌파 급락 쇄도합니다. 평균 궤도 잔차 오류 통계 이면 결산치 역시도 수위 단위 지점 $M = 5$ 궤도상 단위 부분 선에서 단연 뚜렷 확고 선명하고 꽉 막힌 거대 절저 최저 기착 거점(clear minimum) 무사 함몰 도달 안착 성취 기염을 대변 공시 대변합니다.
맞붙은 우측 화면 분할 패널 비교 도면 그림 6.19 진영 대치 뷰는 이번에도 편향 능선 회귀 기틀과 무단 절단 단절 숙청 제재 압박 조교 진영 라쏘 체계들을, 위 동일 일원 편애 조작 집중 환경 데이터상 다시 평행 대치 구동시켜 이들 간 수차 산출 파워 대항 비교 측정 결산 나열 나열선입니다. 비교 전쟁에 참전 맞붙은 세 진영 분석 무리 모델들 그 누가 보아도, 초창기 허술 무구한 차원 단순 최소 구식 제곱 투박 무식 회귀 시절보다 확연 엄청난 위력 파괴 유의미 단계성 거대 쟁취 개선치 표본 결산 향상점을 놀랍게 자랑 시위 내보입니다. 허나, 이번 이 파워 접전 우위 비교 승부 대면 진영에서만은 근소 아주 미세하게도 차원 파편 수합 묶음 PCR 체제 압축 수법 연합 무리 진영과 축 억압 조율 기법 진영 능선 제재 세력군이! 저 독단 0 삭제 라쏘 처단 모델 제어 측정 지표보다도 아주 근소 아슬한 우위 미세 측정 편파 비율 격차 우위 성과로 조금 실적 살며시 능가 퍼포먼스 기록 측정 승리 수치 대반전 쾌거 선방 측면 지수 우세 위력을 판가름 결국 보여 타전 증언해줍니다(slightly outperform).

We note that even though PCR provides a simple way to perform regression using $M < p$ predictors, it is _not_ a feature selection method. This is because each of the $M$ principal components used in the regression is a linear combination of all $p$ of the _original_ features. 우리는 비록 PCR 모델 통제가 비대한 전체 변수 $p$ 개를 죄다 쓰지 않고 줄어든 투입 단면 차원 수량, 즉 $M < p$ 개의 타겟 변수 궤도 부품 단위만을 영리하게 선별 동원하여 회귀를 달성하게 해 주는 심플하고 명확한 가교를 제공하지만, 여기서 분명히 착각하지 말아야 할 경고가 있습니다. 이 치트키 수법이 원본 변수의 속성을 아예 솎아 잘라 버리는 잔혹한 기법인 '특징 선별 방법(feature selection method)' 부류 수단으로는 절대 결코 _아니라는(not)_ 명제 사실을 뼈저리게 유념해야 합니다.
그 자명한 까닭은 예측 모델에 영입 사용된 각 누적 $M$ 개들의 압축 체제 주성분 녀석들 속내를 까보면, 결국 모조리 본래 오리지널(_original_) 변수들 전체 $p$ 개를 하나도 버림 없이 다 엮고 비빔 혼합 비율로 재설계한 선형 결합(linear combination) 합작 산물 덩어리일 따름이기 때문입니다.

원본 초기 핵심 변수 자원들의 추려진 소규모 정예 표본 세트에만 기대 구동하는 압축 제한 모형 체계 단위 시스템의 단면 진전이나 수립 도달 단계 종착지 결과로는 시스템상 수렴해 절대 연결 귀결 도달되지 않는다는 모순 한계 사실입니다.

사실 수식 원리를 까면 증명되듯 무수한 PCR 체재 제어 단면 방식 기저와 능선 회귀 제재 구조 수리 기저는 가히 심도 친밀하게 끈적 동일 밀접 연관 치밀하게 얽혀 수립된 동일 지배 공생 체계 원류에 서 있습니다. 우리는 극단으로 나아가 능선 회귀 통제 제재 튜닝 기법을, 거센 끊김 없이 단절 한계 이면 없이 매끄럽게 연속 전개되는 하나의 부드러운 전면 변수 PCR 체계 단계의 초 무결점 진화 궁극체 무한 전위 "연속 교차 버전(continuous version)" 진화 매개체로 환원 등치시켜 동시 간주해 동일시 판독 해석 처리해 버릴 수도 있는 단계입니다![8]

In PCR, the number of principal components, $M$, is typically chosen by cross-validation. The results of applying PCR to the `Credit` data set are shown in Figure 6.20; the right-hand panel displays the cross-validation errors obtained, as a function of $M$. On these data, the lowest cross-validation error occurs when there are $M = 10$ components; this corresponds to almost no dimension reduction at all, since PCR with $M = 11$ is equivalent to simply performing least squares. 
실무 PCR 과정에서 장착할 주성분의 최적 개수 $M$ 은 일반적으로 **교차 검증(cross-validation)** 평가 점수에 따라 선택됩니다. `Credit` 데이터 세트에 PCR을 적용한 결과가 그림 6.20에 나와 있습니다. 오른쪽 그림은 주성분 개수 $M$ 의 함수로 얻어진 교차 검증 예측 오류를 보여줍니다. 이 데이터의 경우, 주성분이 $M = 10$ 개일 때 가장 낮은 최저 교차 검증 에러 점수가 나타났습니다. 그런데 총 변수가 11개인 데이터에서 $M = 11$ 이면 모든 변수를 다 넣은 기본 최소 제곱 회귀와 동일하므로, $M = 10$ 이라는 이 결과는 사실상 분석에서 차원 축소가 거의 전혀 이뤄지지 않았음을 의미합니다.

When performing PCR, we generally recommend _standardizing_ each predictor, using (6.6), prior to generating the principal components. This standardization ensures that all variables are on the same scale. In the absence of standardization, the high-variance variables will tend to play a larger role in the principal components obtained, and the scale on which the variables are measured will ultimately have an effect on the final PCR model. However, if the variables are all measured in the same units (say, kilograms, or inches), then one might choose not to standardize them. 
PCR을 수행할 때 한 가지 핵심 조언을 명심하세요. 처음 주성분들을 생성해내기 전에 예측 변수들을 식 (6.6)에 따라 무조건 **표준화(standardizing)** 할 것을 강력히 권장합니다. 이렇게 해야 모든 변수들이 페널티 없이 다 같이 동일한 크기(scale) 안에서 경쟁하도록 보장됩니다. 만약 이 표준화를 제대로 안 한다면, 겉보기에 큰 측정 수치의 덩치만 큰 이른바 고분산 측정 변수(high-variance variables)들이 도출된 주성분 안에서 가장 큰 비중(larger role)을 부당하게 차지하고 장악해버리는 경향이 발생합니다. 그 결과, 각 변수의 겉보기 수치 배율만이 최종 PCR 회귀 모델 결과 전체를 결정짓고 망쳐버리는 재앙을 낳게 됩니다. 그러나 예외적으로 원래 데이터 변수들이 (예를 들어 전부 다 '킬로그램' 이나 '인치' 등) 하나같이 처음부터 다 똑같은 하나의 동일한 단위 규격으로 통일 측정된 거라면, 통계 모델러 재량에 따라 억지로 귀찮게 표준화를 시킬 필요는 없을 수도 있습니다.
