# 6.5.1 부분집합 선택 기법 (Subset Selection Methods)

Here we implement methods that reduce the number of parameters in a model by restricting the model to a subset of the input variables.
이번 장에서 우리는 모델 구성을 투입 입력 변수들의 특정한 부분집합으로 강제 제한시킴으로써, 모델 내에 존재하는 전체 파라미터 계수의 개수를 억제하고 줄여나가는 기법들을 실질적으로 코드로 구현해 봅니다.
