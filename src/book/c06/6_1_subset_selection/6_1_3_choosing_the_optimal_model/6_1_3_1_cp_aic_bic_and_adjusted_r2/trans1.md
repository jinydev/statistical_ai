# $C_p$, AIC, BIC, and Adjusted $R^2$ 
# $C_p$, AIC, BIC 및 조정된 $R^2$ ($C_p$, AIC, BIC, and Adjusted $R^2$)

We show in Chapter 2 that the training set $\text{MSE}$ is generally an underestimate of the test MSE. (Recall that $\text{MSE}$ = RSS $/n$.) This is because when we fit a model to the training data using least squares, we specifically estimate the regression coefficients such that the training RSS (but not the test RSS) is as small as possible. In particular, the training error will decrease as more variables are included in the model, but the test error may not. Therefore, training set RSS and training set $R^2$ cannot be used to select from among a set of models with different numbers of variables. 
제 2장에서 보았듯이 훈련 세트의 $\text{MSE}$는 일반적으로 테스트 $\text{MSE}$를 과소평가합니다. (여기서 $\text{MSE} = \text{RSS} / n$ 입니다.) 이는 최소 제곱법으로 모델을 훈련 데이터에 적합시킬 때 훈련 $\text{RSS}$가 가능한 한 작아지도록 예측 계수를 추정하기 때문입니다. 특히 관련 변수가 추가될수록 모델의 훈련 오차는 확실히 감소하지만 테스트 오차는 그렇지 않을 수 있습니다. 따라서 모델의 변수 개수가 전혀 다를 때, 훈련 $\text{RSS}$ 및 관측 변수 $R^2$ 는 최적 모델을 지정 선택하는 주요 판단 기준으로 사용할 수 없습니다.

However, a number of techniques for _adjusting_ the training error for the model size are available. These approaches can be used to select among a set of models with different numbers of variables. We now consider four such approaches: $C_p$, Akaike information criterion (AIC), Bayesian information criterion (BIC), and adjusted $R^2$. Figure 6.2 displays $C_p$, BIC, and adjusted $R^2$ for the best model of each size produced by best subset selection on the `Credit` data set. 
그러나 다행히도, 모델의 전체 크기 인자를 추가 반영하여 학습 오차를 수학적으로 '조정(adjusting)' 시키는 유효한 측정 기법들이 다수 존재합니다. 이 해당 평가 방식들을 직접 사용하면 지정 변수 개수가 상호 다른 모델 세트 그룹 중에서 단일 최상위 챔피언 모델을 발탁 지정할 수 있습니다. 이제 우리는 가장 기초적인 4대장인 $C_p$, 아카이케 주요 정보 도구 식(AIC), 베이즈 주요 정보 도구 식(BIC), 그리고 별도 최적 조정된 $R^2$ 라는 4가지 판단 잣대 접근법을 순차적으로 살펴볼 예정입니다. 그림 6.2는 대표 `Credit` 타겟 데이터 세트에 관련 최적 서브 부분집합 측정 기법을 전부 사용하여 도출해 낸 각 모델 변수 개수 크기별 에이스 모델들의 $C_p$, BIC, 추가 조정된 $R^2$ 평가 지표 수치들을 보여줍니다.

For a fitted least squares model containing $d$ predictors, the $C_p$ estimate of test $\text{MSE}$ is computed using the equation
$d$ 개의 단일 예측 변수를 전체 포함하도록 사전에 피팅 적합된 최소 오차 제곱 모델에 대하여, 테스트 $\text{MSE}$ 에 대한 $C_p$ 에러 추정치는 다음 공식을 적용하여 연산 계산됩니다:

$$
C_p = \frac{1}{n} (\text{RSS} + 2 d \hat{\sigma}^2) \quad (6.2)
$$

where $\hat{\sigma}^2$ is an estimate of the variance of the error $\epsilon$ associated with each response measurement in (6.1).[^4] Typically $\hat{\sigma}^2$ is estimated using the full model containing all predictors. Essentially, the $C_p$ statistic adds a penalty of $2 d \hat{\sigma}^2$ to the training RSS in order to adjust for the fact that the training error tends to underestimate the test error. Clearly, the penalty increases as the number of predictors in the model increases; this is intended to adjust for the corresponding decrease in training RSS. Though it is beyond the scope of this book, one can show that if $\hat{\sigma}^2$ is an unbiased estimate of $\sigma^2$ in (6.2), then $C_p$ is an unbiased estimate of test MSE. As a consequence, the $C_p$ statistic tends to take on a small value for models with a low test error, so when determining which of a set of models is best, we choose the model with the lowest $C_p$ value. In Figure 6.2, $C_p$ selects the six-variable model containing the predictors `income` , `limit` , `rating` , `cards` , `age` and `student` . 
이 수식에서 $\hat{\sigma}^2$ 은 모델 (6.1)의 오차항 성분 $\epsilon$ 요소에 대한 전체 분산 측정 추정치입니다.[^4] 가장 일반적으로 이 변수 분산 $\hat{\sigma}^2$ 지정 값은 전체 활용 예측 변수를 전체 포함한 최대 전체 모델(full model)을 집중 사용하여 계산 추정합니다. 가장 본질적으로, $C_p$ 결과 통계량은 지정 훈련 오차가 평가 테스트 오차를 극심하게 과소평가하는 부정적 현상을 수학적으로 바로잡기 위해 $2 d \hat{\sigma}^2$ 이라는 추가 페널티(penalty) 처분 수량을 기존 훈련 $\text{RSS}$ 항목에 추가 부과합니다. 해당 모델에 적용 포함된 예측 인자 변수 개수 측량치 $d$ 가 점점 늘어날수록 적용 페널티 벌점 또한 비례하여 수직 상승 증가하며; 이 부분은 변수 추가에 따라 기계적으로 훈련 $\text{RSS}$ 가 하락 감소하는 부정적 현상을 인위적으로 무마하려는 의도입니다. 이 서적의 지정 난이도를 완전 벗어나지만, 식별 식 (6.2)의 분산 $\hat{\sigma}^2$ 가 참된 $\sigma^2$ 의 절대 비편향 추정치일 경우 $C_p$ 공식 역시 테스트 $\text{MSE}$ 에 대한 비편향 정상 추정치가 됨이 증명되었습니다. 가장 결정적으로, 적용 테스트 오차가 최저 낮은 핵심 모델일수록 $C_p$ 산출 지표 역시 가장 작은 마진 값을 보유 나타내는 규칙 경향이 생기므로, 가장 수치 $C_p$ 값이 낮은 단일 모델을 승자 모델로 지정 선택하는 것이 정답입니다. 본문 그림 6.2의 측정 $C_p$ 평가 지표는 요소 변수 총 6개를 전격 포함하는 6-변수 단일 모델 구조를 최종 우승 에이스 모델로 지목합니다.

> [^4] 본 멜로우즈 통계 학자의 정식 $C_p$ 지수 평가는 관련 현장 업무에서 종종 $C_p' = \text{RSS} / \hat{\sigma}^2 + 2d - n$ 수식 형태인 변형식 구조로 지정 정의되기도 합니다. 이 부분은 수학적 구조 풀이 관점에서 본문에 최초 제시된 기준 공식 규칙과 $C_p = \frac{1}{n}\hat{\sigma}^2(C_p' + n)$ 치환 형태 식 측면에서 완전 물리적으로 동일 일치합니다. 고로 가장 작은 단일 $C_p$ 지수 단위 값을 지닌 단독 선출 모델 개체는 항상 무조건 항상 가장 최저점 최소 크기 $C_p'$ 지표 결과 값을 똑같이 산출 가지게 성립됩니다.

The AIC criterion is defined for a large class of models fit by maximum likelihood. In the case of the model (6.1) with Gaussian errors, maximum likelihood and least squares are the same thing. In this case AIC is given by
그 이외의 평가 척도 AIC(아카이케 주요 정보 도구 기준)는 최대 타겟 우도법 연산으로 훈련된 매우 광범위한 다수 수리 모델 구조들에 전체 걸쳐 범용 정의됩니다. 수학적 가우시안 곡선 오차 패턴을 지닌 전체 모델 (6.1) 평가 환경 상황 하에서는, 최대 특정 우도법 기법과 최소 오차 제곱법 구조 방식의 수리 연산이 절대적으로 동일한 똑같은 산출 결과를 발생 도출합니다. 그리하여 이 분석 경우 AIC 수식 체계는 다음과 같습니다.

$$
\text{AIC} = \frac{1}{n \hat{\sigma}^2} (\text{RSS} + 2 d \hat{\sigma}^2) \quad (6.3)
$$

where, for simplicity, we have omitted irrelevant constants.[^5] Hence for least squares models, $C_p$ and AIC are proportional to each other, and so only $C_p$ is displayed in Figure 6.2. 
여기 6.3 식의 가독 단순화를 목적으로, 수식 풀이 해석에 전혀 필요하지 않은 잉여 무관 부가 상수들은 사전에 전체 의도적으로 제거 배제했습니다.[^5] 고전 최소 오차 제곱 선형 모델 상황에서는 필수적으로 이 $C_p$ 지표 단위와 AIC 단위 수치가 상호 비례(proportional) 되는 동등 관계에 영원히 묶입니다. 그리하여 첨부 그림 6.2 패널에서는 굳이 번거롭게 AIC 지표 항목을 따로 표기 표시하지 아니하고 단지 $C_p$ 지표 평가 값 그래프 하나 단독으로 대표 노출 표시하였습니다.

> [^5] 클래식 최소 오차 제곱 공식 회귀 분석 연산을 도입한 AIC 수리 계산 기본 공식 구조에는 통상 두 종류 가지 공식 버전이 따로 존재합니다. 우선 첫째 첫 버전 (6.3) 단위 차트 공식 구조는 활용 모든 타겟 예측 변수가 전체 몽땅 포함 집중된 풀-사이즈 전체 모델 단위를 직접 사용해 먼저 도출해 낸 선행 $\sigma^2$ 오차 분산의 예측 추정치 지수를 수리적으로 요구합니다. 그에 대조되는 추가 두 번째 식 버전 구조 공식은 이 $\sigma^2$ 분산 수치가 사전 미지수 조건이고 또한 고의 명시적 사전 추정을 아예 건너뛰고 배제하고 싶을 당시에 수리 적합하며; 이 때 해당 수리 계산 식이 사전 $\text{RSS}$ 평가 단순항 부위 대신 특별 $\log(\text{RSS})$ 연산 항을 대체 심어 사용합니다. 해당 이들 두 특이 공식들에 얽힌 깊숙한 상세 수리 연산 구조 유도 도출 전개 과정 기전은 전적으로 책의 허용 범주 교육 진도를 완전히 이탈하므로 구체 설명 생략합니다.

BIC is derived from a Bayesian point of view, but ends up looking similar to $C_p$ (and AIC) as well. For the least squares model with $d$ predictors, the BIC is, up to irrelevant constants, given by
또 다른 세 번째 평가 지표 척도 수단 BIC(베이즈 수리 정보 도구 기준)는 철저히 이질적인 학파인 '베이즈 관점'에서 새롭게 수리 유도되었음에도, 최종적으로 도출 도달한 수식 방정식 겉면 모형 외형과 형태 구조는 과거 기존 구형 $C_p$ 및 신형 AIC 와 무척 엄청 놀라울 만큼 유사하게 귀결 흡사합니다. 대상 $d$ 단위 묶음 예측 변수를 구성 지닌 전통 최소 오차 제곱 선형 모델 세트에 한정 대응하여 수리적으로 불필요 떨거지 꼬리 상수를 полностью 제거 처리한 BIC 계산 방정식 구조 결과는 다음과 같이 주어집니다.

$$
\text{BIC} = \frac{1}{n} (\text{RSS} + \log(n) d \hat{\sigma}^2) \quad (6.4)
$$

Like $C_p$, the BIC will tend to take on a small value for a model with a low test error, and so generally we select the model that has the lowest BIC value. Notice that BIC replaces the $2 d \hat{\sigma}^2$ used by $C_p$ with a $\log(n) d \hat{\sigma}^2$ term, where $n$ is the number of observations. Since $\log n > 2$ for any $n > 7$, the BIC statistic generally places a heavier penalty on models with many variables, and hence results in the selection of smaller models than $C_p$. In Figure 6.2, we see that this is indeed the case for the `Credit` data set; BIC chooses a model that contains only the four predictors `income` , `limit` , `cards` , and `student` . In this case the curves are very flat and so there does not appear to be much difference in accuracy between the four-variable and six-variable models. 
과거 $C_p$ 지수처럼, 신생 BIC 잣대 평가 지표 역시 적용 테스트 검증 오차가 낮은 최적 모델일수록 작고 낮아진 스코어 평가점수 값을 기록 나타내는 수리적 성질 부분이 동일하게 있으므로, 분석가 우리는 단순히 전적으로 가장 작고 밑바닥 찌그러진 최저값 BIC 판독 달성 모델 녀석을 선별 택하게 귀결됩니다. 핵심 주목할 극적 요인 차이점 요소는 BIC 평가 기재 기법이 과거 기존 $C_p$ 공식 에 사전 존재 정착 하던 페널티 벌점 측정 항구조 $2 d \hat{\sigma}^2$ 단위 항목을 완전히 잘라 버리고, 그 빈 수리 자리에 이보다 훨씬 잔혹하고 더 강력한 $\log(n) d \hat{\sigma}^2$ 페널티 폭탄 벌점 척도 항 기재로 대체(replaces) 적용 활용했다는 확실한 사실 현상입니다. 여기서 분석 샘플 총 데이터 관측치 개당 샘플 개수 단위 규모 측정 $n$ 수 숫자가 고작 7 단위 선을 살금 상회 아주 약간만 돌파 넘기기($n > 7$) 만 단연 하여도 수리 부등식 요건인 $\log n > 2$ 구조 상태가 수학적으로 언제나 상시 확정 성립하게 이루어집니다.
결과적으로 도출 이는 지표 BIC 지표 통계 점수 지표가 변수 인원 체급 개수가 굉장히 비체 비대해진 몸집이 큰 다 변수 뚱뚱한 적용 모델들에게 지난 과거 기존 구식 평가 지표들 도구 장벽 보다 비교할 수 없이 훨씬 더 잔혹 흉포하게 짓누르는 거대 무거운 페널티(penalty) 제재 치명 벌점을 냅다 가격 부과하기 때문으로 파생되며, 이는 종착 결국 $C_p$ 잣대 평가망 체계 심사 기준 대비 훨씬 적고 소규모 모델 구성 인원 변수 개수 조각 풀이 더 축소 적은, 극히 컴팩트 제한적 단소 다이어트 체형 소형 모델 대상을 최고 1등 우승 챔피언 모델 파벌 우승 모델 무리로 채택 발탁 선출(selection) 하게 될 자연스러운 개연성과 확률이 기하급수적으로 막대 매우 높아지는 수학 모델 결론을 이룩 양산 반환 도출 낳습니다. 확인 위 패널 그림 6.2 도판 데이터 해당 `Credit` 정보 데이터 세트 성능 심사 성적 그래프 도면이 이 거대한 반전 현상 사실을 직접 눈에 띠게 사실 명시 증거 증명 증빙합니다; 이 상황 가혹한 벌점 체제 BIC 통제 평정 체제 기준 치수는 적용 모델 변수를 겨우 제한적 4명만 확보 탑재 소유 보유한 4-변수 소형 절약 축소 미니 다이어트 제어 모델 진영을 1위 우승 에이스 단일 챔피언 우등 타겟 으로 편애 채용 발탁 임명 지목 선출(chooses) 처리 했습니다. 다만 현재 해당 해당 단면 도면 위 분석 그래프 곡선 전개 추이 라인은 이미 4-변수 추가 진입 구간을 지난 해당 케이스 상황 지점 이후 도처 몹시 굉장히 매우 평탄 고원 지루 차이 없는 고요(flat) 양상을 달려 유지 펼치 형성 그려 나아 가기 때문에 결론 차원 4-변수나 6-변수 개체 모델 체계 상호 등급 간 에는 성능 적중 정교도 오차 우열(accuracy) 관련 사실상 체감 눈에 뚜렷 확연 선명 대대 대단 심각 심히 심대 압도 막대 거창 거대 (much) 띄는 수치 질적 큰 격차 오차 편견 스펙 차등 간극 차이 패권 차이(difference) 양상은 거의 절대 아예 없는 무존 차이 수렴 전무 부재(does not appear) 한 실상 상태 것으로 실질 관측 육안 분석 파악 관전 도출 인지 됩니다.

The adjusted $R^2$ statistic is another popular approach for selecting among a set of models that contain different numbers of variables. Recall from Chapter 3 that the usual $R^2$ is defined as $1 − \text{RSS} / \text{TSS}$, where $\text{TSS} = \sum(y_i − \bar{y})^2$ is the _total sum of squares_ for the response. Since RSS always decreases as more variables are added to the model, the $R^2$ always increases as more variables are added. $$
\text{Adjusted } R^2 = 1 - \frac{\text{RSS} / (n - d - 1)}{\text{TSS} / (n - 1)} \quad (6.5)
$$

Unlike $C_p$, AIC, and BIC, for which a _small_ value indicates a model with a low test error, a _large_ value of adjusted $R^2$ indicates a model with a small test error. Maximizing the adjusted $R^2$ is equivalent to minimizing $\frac{\text{RSS}}{n−d−1}$. While RSS always decreases as the number of variables in the model increases, $\frac{\text{RSS}}{n−d−1}$ may increase or decrease, due to the presence of $d$ in the denominator.

The intuition behind the adjusted $R^2$ is that once all of the correct variables have been included in the model, adding additional _noise_ variables will lead to only a very small decrease in RSS. Since adding noise variables leads to an increase in $d$, such variables will lead to an increase in $\frac{\text{RSS}}{n−d−1}$, and consequently a decrease in the adjusted $R^2$. Therefore, in theory, the model with the largest adjusted $R^2$ will have only correct variables and no noise variables. Unlike the $R^2$ statistic, the adjusted $R^2$ statistic _pays a price_ for the inclusion of unnecessary variables in the model. Figure 6.2 displays the adjusted $R^2$ for the `Credit` data set.
