# 6.2.3 조율 파라미터의 선택 (Selecting the Tuning Parameter) 

Just as the subset selection approaches considered in Section 6.1 require a method to determine which of the models under consideration is best, implementing ridge regression and the lasso requires a method for selecting a value for the tuning parameter $\lambda$ in (6.5) and (6.7), or equivalently, the value of the constraint $s$ in (6.9) and (6.8). Cross-validation provides a simple way to tackle this problem. We choose a grid of $\lambda$ values, and compute the cross-validation error for each value of $\lambda$, as described in Chapter 5. We then select the tuning parameter value for which the cross-validation error is smallest. Finally, the model is re-fit using all of the available observations and the selected value of the tuning parameter. 
6.1절에서 고려한 부분집합 선택 방식이 고려 대상 중 어떤 모델이 가장 우수한지 결정할 수단을 요구하는 것처럼, 릿지 회귀와 라쏘를 실행하기 위해서도 식 (6.5)와 (6.7)의 조율 파라미터 $\lambda$ 값을 선택할 수단이 요구되며, 이는 수식 (6.9)와 (6.8)의 제약 조건 $s$ 값을 선택하는 것과 완전히 동일합니다. 교차 검증은 이 문제를 다루는 직관적이고 쉬운 방법을 제공합니다. 우리는 여러 값들로 구성된 $\lambda$ 그리드를 설정하고, 5장에서 배운대로 각 $\lambda$ 값에 대한 교차 검증 오차를 계산합니다. 그런 다음 교차 검증 오차가 가장 최저가 되는 튜닝 파라미터 값을 최종 선택합니다. 마지막으로, 사용 가능한 전체 관측치 데이터와 방금 선별된 튜닝 파라미터 값을 사용하여 최종 모델을 다시 적합(re-fit) 합니다.

Figure 6.12 displays the choice of $\lambda$ that results from performing leave-one-out cross-validation on the ridge regression fits from the `Credit` data set. The dashed vertical lines indicate the selected value of $\lambda$. In this case the value is relatively small, indicating that the optimal fit only involves a small amount of shrinkage relative to the least squares solution. In addition, the dip is not very pronounced, so there is rather a wide range of values that would give a very similar error. In a case like this we might simply use the least squares solution. 
그림 6.12는 `Credit` 데이터 세트에 릿지 회귀 모델을 적용하고 단일 관측치 제거(LOOCV) 교차 검증 기법을 수행하여 도출된 $\lambda$ 선별 결과를 보여줍니다. 수직 점선은 여기서 선택된 $\lambda$ 의 값을 가리킵니다. 이 경우 $\lambda$ 수치값은 상당히 작은 편에 속하며, 이는 최적 타겟 적합 모델이 최소 제곱 해 공식에 비교했을 때 겨우 미세한 수준의 계수 수축 작업만을 요구하고 있음을 의미합니다. 게다가 이 오차 곡선의 강하 구간(dip) 또한 무척이나 완만하여 뚜렷하지 않기 때문에, 사실상 거의 유사한 오차율 수준을 보장해줄 수 있는 $\lambda$ 값의 허용 범위가 상당히 넓게 분포하고 있습니다. 이 같은 특수 상황이라면 우리는 차라리 심플하게 일반 최소 제곱 해답을 그냥 고수하여 사용해볼 수도 있겠습니다.

Figure 6.13 provides an illustration of ten-fold cross-validation applied to the lasso fits on the sparse simulated data from Figure 6.9. The left-hand panel of Figure 6.13 displays the cross-validation error, while the right-hand panel displays the coefficient estimates. The vertical dashed lines indicate the point at which the cross-validation error is smallest. The two colored lines in the right-hand panel of Figure 6.13 represent the two predictors that are related to the response, while the grey lines represent the unrelated predictors; these are often referred to as _signal_ and _noise_ variables, respectively. Not only has the lasso correctly given much larger coefficient estimates to the two signal predictors, but also the minimum cross-validation error corresponds to a set of coefficient estimates for which only the signal variables are non-zero. Hence cross-validation together with the lasso has correctly identified the two signal variables in the model, even though this is a challenging setting, with $p = 45$ variables and only $n = 50$ observations. In contrast, the least squares solution—displayed on the far right of the right-hand panel of Figure 6.13—assigns a large coefficient estimate to only one of the two signal variables. 
그림 6.13은 앞서 6.9에서 등장했던 희소(sparse) 시뮬레이션 합성 데이터 환경 하에 구축된 라쏘 모형 적합물에 10-겹 교차 검증 장치를 런칭시킨 예시 국면을 전시해줍니다. 그림 6.13의 왼쪽 측면 패널 구간은 교차 검증 오류 점수를 보여주며, 오른쪽 측면 패널 파트는 모델 파라미터 계수 추정치를 드러내 나타냅니다. 수직으로 우뚝 선 점선은 교차 검증 오차 기록이 가장 최저로 갱신된 해당 점의 지점을 명백히 지목합니다. 그림 6.13 우측 패널 국면 위로 색칠된 뚜렷한 두 실선 모양새 궤도들은 종속 시스템 반응과 밀접히 연관성이 입증된 2개의 핵심 예측 변수를 묘사하는 반면, 회색 점선들은 결과와 하등 무관한 쓸데없는 나머지 잉여 예측 변수들을 대변합니다; 이들은 종종 통계학에서 순서대로 무수한 _신호(signal)_ 와 단순 _잡음(noise)_ 변수로 칭해집니다. 이 라쏘 기법 체제는 심히 감탄스럽게도 2가지 기둥 신호 핵심 예측 변수들에게 확실히 거대한 파라미터 추정 수치를 한껏 배정 할당 제공 하였을 뿐만 아니라, 그 최소점 기록인 교차 검증 오류 좌표의 결론이 기가 막히게도 정확히 두 신호 변수만이 0이 아닌 형태로 생존한 계수 집단 모델에 부합 맞아 떨어진다는 흥미로운 분석 결과를 선사합니다. 고로 훌륭한 라쏘와 교차 검증 장치가 결속 맺은 이 탁월한 합작 성과는, 관측 대상 $n = 50$ 개에 매몰되어 $p = 45$ 차원 변수가 이중고를 가져다주는 매우 도전적 역경 한계 세팅 속임에도 불구하고 모델 깊숙한 곳에서 중요 2단 핵심 신호 변수 실체를 정확하게 인식 지목 간파 식별(identified) 하여 잡아내는 엄청난 성과를 남깁니다. 이와 극명하게 반대 대조를 이루며 그림 6.13 패널 구간 맨 제일 구석 우측 구역에 기재 묘사 도표 전시된 기존 방어 기저 최소 제곱 모형 측면 결론은 안타깝게도 두 핵심 신호 파장 변수 중에 단 1명에게만 그 막대한 파라미터 계수 분량을 오조준하여 일괄 배정 할당하고 만다는 씁쓸한 한계 결론을 드러냅니다.
